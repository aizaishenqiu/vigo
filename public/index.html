MVC 架构

  • Controller-Service-Model 清晰分层
  • IoC 容器依赖注入
  • 完整中间件链支持
  • 灵活的路由系统
  • 内置模板引擎
🗄️

ThinkPHP/Laravel风格 ORM

  • 链式操作查询构造器
  • 模型关联（HasOne/HasMany）
  • 多数据库支持（MySQL/PG/SQLite/MSSQL）
  • 读写分离 + 分库分表
  • 乐观锁 + 分布式锁
🌐

微服务架构

  • gRPC 高性能 RPC 框架
  • Nacos 服务注册与发现
  • 配置中心（Nacos/Apollo）
  • 负载均衡（4 种算法）
  • 熔断降级 + 链路追踪
⚡

多级缓存系统

  • BigCache 内存缓存
  • Redis 分布式缓存
  • 多级缓存 + 标签管理
  • 缓存预热 + 自动刷新
  • 缓存穿透/雪崩防护

消息队列
