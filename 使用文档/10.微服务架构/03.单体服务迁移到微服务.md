## 1. 迁移前准备

### 1.1 评估是否需要迁移

**✅ 需要迁移的信号**：

| 问题       | 表现                 | 影响           |
| ---------- | -------------------- | -------------- |
| 代码库过大 | 编译时间 > 5 分钟    | 开发效率低     |
| 部署困难   | 每次部署整个应用     | 风险高、耗时长 |
| 性能瓶颈   | 某个模块占用大量资源 | 无法独立扩展   |
| 技术栈限制 | 无法使用新技术       | 创新受阻       |
| 团队规模大 | 代码冲突频繁         | 协作困难       |

**❌ 暂不需要迁移的情况**：

- 团队规模 < 10 人
- 业务相对简单
- 日活用户 < 10 万
- 没有运维团队支持

### 1.2 迁移策略选择

#### 策略一：绞杀者模式（推荐）

```
逐步在单体边缘构建新服务，最终替换单体

单体应用 → 单体 + 用户服务 → 单体 + 用户 + 订单服务 → 完全微服务化
```

#### 策略二：并行运行模式

```
新旧系统并行运行，通过流量切换逐步迁移

单体应用 (100% 流量) → 微服务 (10% 流量) → 微服务 (50% 流量) → 微服务 (100% 流量)
```

#### 策略三：一次性拆分

```
一次性完成所有服务拆分（风险高，不推荐）

单体应用 → 微服务集群
```

```go
package main

import (
    "context"
    "flag"
    "fmt"
    "log"
    "os"
    "os/signal"
    "syscall"
    "time"

    "user-service/internal/handler"
    "user-service/internal/repository"
    "user-service/internal/service"

    "vigo/framework"
    "vigo/framework/config"
    "vigo/framework/db"
    "vigo/framework/discovery"
    "vigo/framework/log"
    "vigo/framework/mvc"
    "vigo/framework/trace"
)

func main() {
    // 解析命令行参数
    configPath := flag.String("config", "config/config.yaml", "配置文件路径")
    flag.Parse()

    // 加载配置
    cfg, err := config.Load(*configPath)
    if err != nil {
        log.Fatalf("加载配置失败：%v", err)
    }

    // 初始化日志
    log.Init(log.Config{
        Level:  cfg.Log.Level,
        Format: cfg.Log.Format,
        Output: cfg.Log.Output,
    })

    log.Log.Infof("启动 %s v%s", cfg.App.Name, cfg.App.Version)

    // 初始化数据库
    if err := initDB(cfg); err != nil {
        log.Fatalf("初始化数据库失败：%v", err)
    }

    // 初始化链路追踪
    if cfg.Trace.Enabled {
        closer, err := trace.Init(trace.Config{
            Type:      cfg.Trace.Type,
            Endpoint:  cfg.Trace.Endpoint,
            Service:   cfg.App.Name,
            SampleRate: cfg.Trace.SampleRate,
        })
        if err != nil {
            log.Fatalf("初始化追踪失败：%v", err)
        }
        defer closer.Close()
    }

    // 初始化服务发现
    if err := initDiscovery(cfg); err != nil {
        log.Fatalf("初始化服务发现失败：%v", err)
    }

    // 创建 MVC 应用
    app := mvc.New()

    // 注册路由
    registerRoutes(app)

    // 启动 HTTP 服务器
    go func() {
        addr := fmt.Sprintf(":%d", cfg.App.Port)
        log.Log.Infof("HTTP 服务器启动：%s", addr)
        if err := app.Run(addr); err != nil {
            log.Fatalf("启动 HTTP 服务器失败：%v", err)
        }
    }()

    // 等待退出信号
    quit := make(chan os.Signal, 1)
    signal.Notify(quit, syscall.SIGINT, syscall.SIGTERM)
    <-quit

    log.Log.Info("正在关闭服务器...")

    // 优雅关闭
    ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
    defer cancel()

    if err := app.Shutdown(ctx); err != nil {
        log.Fatalf("关闭服务器失败：%v", err)
    }

    log.Log.Info("服务器已关闭")
}

// initDB 初始化数据库
func initDB(cfg *config.Config) error {
    return db.Init(db.Config{
        Driver:          cfg.Database.Driver,
        DSN:             cfg.Database.DSN,
        MaxOpenConns:    cfg.Database.MaxOpenConns,
        MaxIdleConns:    cfg.Database.MaxIdleConns,
        ConnMaxLifetime: time.Duration(cfg.Database.ConnMaxLifetime) * time.Second,
    })
}

// initDiscovery 初始化服务发现
func initDiscovery(cfg *config.Config) error {
    return discovery.Init(discovery.Config{
        Type:        cfg.Discovery.Type,
        ServerAddr:  cfg.Discovery.ServerAddr,
        NamespaceID: cfg.Discovery.NamespaceID,
        Group:       cfg.Discovery.Group,
    })
}

// registerRoutes 注册路由
func registerRoutes(r *mvc.Router) {
    // 创建处理器
    userHandler := handler.NewUserHandler(
        service.NewUserService(repository.NewUserRepository()),
    )

    // 健康检查
    r.GET("/health", func(c *mvc.Context) {
        c.Json(200, map[string]interface{}{
            "status": "ok",
            "time":   time.Now().Format(time.RFC3339),
        })
    })

    // 用户路由
    v1 := r.Group("/api/v1")
    {
        v1.GET("/users", userHandler.List)
        v1.GET("/users/:id", userHandler.Detail)
        v1.POST("/users", userHandler.Create)
        v1.PUT("/users/:id", userHandler.Update)
        v1.DELETE("/users/:id", userHandler.Delete)
        v1.POST("/auth/login", userHandler.Login)
        v1.POST("/auth/register", userHandler.Register)
    }
}
```

```go
package model

import (
    "time"

// User 用户模型
type User struct {
    ID        int64     `json:"id" gorm:"primaryKey"`
    Email     string    `json:"email" gorm:"uniqueIndex;size:100;not null"`
    Password  string    `json:"-" gorm:"size:255;not null"`
    Nickname  string    `json:"nickname" gorm:"size:50"`
    Avatar    string    `json:"avatar" gorm:"size:255"`
    Status    int8      `json:"status" gorm:"default:1"` // 1:正常，0:禁用
    CreatedAt time.Time `json:"created_at" gorm:"autoCreateTime"`
    UpdatedAt time.Time `json:"updated_at" gorm:"autoUpdateTime"`
}

// TableName 指定表名
func (User) TableName() string {
    return "users"
}

// CreateUserRequest 创建用户请求
type CreateUserRequest struct {
    Email    string `json:"email" validate:"required,email"`
    Password string `json:"password" validate:"required,min=6"`
    Nickname string `json:"nickname" validate:"max=50"`
}

// UpdateUserRequest 更新用户请求
type UpdateUserRequest struct {
    Nickname string `json:"nickname" validate:"max=50"`
    Avatar   string `json:"avatar" validate:"max=255"`
}

// LoginRequest 登录请求
type LoginRequest struct {
    Email    string `json:"email" validate:"required,email"`
    Password string `json:"password" validate:"required"`
}

// LoginResponse 登录响应
type LoginResponse struct {
    Token     string    `json:"token"`
    ExpiresAt time.Time `json:"expires_at"`
    User      *User     `json:"user"`
}
```

```go
package repository

import (
    "context"
    "fmt"

    "user-service/internal/model"

)

// UserRepository 用户数据访问
type UserRepository struct {
    db *db.DB
}

// NewUserRepository 创建用户数据访问
func NewUserRepository() *UserRepository {
    return &UserRepository{
        db: db.GetDB(),
    }
}

// Create 创建用户
func (r *UserRepository) Create(ctx context.Context, user *model.User) error {
    return r.db.WithContext(ctx).Create(user).Error
}

// GetByID 根据 ID 获取用户
func (r *UserRepository) GetByID(ctx context.Context, id int64) (*model.User, error) {
    var user model.User
    err := r.db.WithContext(ctx).First(&user, id).Error
    if err != nil {
        return nil, fmt.Errorf("获取用户失败：%w", err)
    }
    return &user, nil
}

// GetByEmail 根据邮箱获取用户
func (r *UserRepository) GetByEmail(ctx context.Context, email string) (*model.User, error) {
    var user model.User
    err := r.db.WithContext(ctx).Where("email = ?", email).First(&user).Error
    if err != nil {
        return nil, fmt.Errorf("获取用户失败：%w", err)
    }
    return &user, nil
}

// Update 更新用户
func (r *UserRepository) Update(ctx context.Context, user *model.User) error {
    return r.db.WithContext(ctx).Save(user).Error
}

// Delete 删除用户
func (r *UserRepository) Delete(ctx context.Context, id int64) error {
    return r.db.WithContext(ctx).Delete(&model.User{}, id).Error
}

// FindPage 分页查询用户
func (r *UserRepository) FindPage(ctx context.Context, page, pageSize int) ([]*model.User, int, error) {
    var total int64
    var users []*model.User

    // 查询总数
    if err := r.db.WithContext(ctx).Model(&model.User{}).Count(&total).Error; err != nil {
        return nil, 0, fmt.Errorf("查询总数失败：%w", err)
    }

    // 查询数据
    offset := (page - 1) * pageSize
    err := r.db.WithContext(ctx).
        Offset(offset).
        Limit(pageSize).
        Order("id DESC").
        Find(&users).Error
    if err != nil {
        return nil, 0, fmt.Errorf("查询用户列表失败：%w", err)
    }

    return users, int(total), nil
}
```

### 3.3 业务逻辑层

**文件**: `user-service/internal/service/user_service.go`

```go
package service

import (
    "context"
    "errors"
    "time"

    "user-service/internal/model"
    "user-service/internal/repository"

    "vigo/framework/cache"
    "vigo/framework/jwt"
    "vigo/framework/log"
    "golang.org/x/crypto/bcrypt"
)

// UserService 用户服务
type UserService struct {
    repo  *repository.UserRepository
    cache cache.Cache
}

// NewUserService 创建用户服务
func NewUserService(repo *repository.UserRepository) *UserService {
    return &UserService{
        repo:  repo,
        cache: cache.NewRedisCache(),
    }
}

// GetList 获取用户列表
func (s *UserService) GetList(ctx context.Context, page, pageSize int) ([]*model.User, int, error) {
    return s.repo.FindPage(ctx, page, pageSize)
}

// GetByID 根据 ID 获取用户
func (s *UserService) GetByID(ctx context.Context, id int64) (*model.User, error) {
    // 先从缓存获取
    cacheKey := fmt.Sprintf("user:%d", id)
    var user model.User

    if err := s.cache.Get(cacheKey, &user); err == nil {
        log.Log.Debugf("从缓存获取用户：%d", id)
        return &user, nil
    }

    // 从数据库获取
    user, err := s.repo.GetByID(ctx, id)
    if err != nil {
        return nil, err
    }

    // 写入缓存（5 分钟过期）
    s.cache.Set(cacheKey, user, 5*time.Minute)
    return user, nil
}

// Create 创建用户
func (s *UserService) Create(ctx context.Context, req *model.CreateUserRequest) (*model.User, error) {
    // 检查邮箱是否已存在
    if _, err := s.repo.GetByEmail(ctx, req.Email); err == nil {
        return nil, errors.New("邮箱已被注册")
    }

    // 密码加密
    hashedPassword, err := bcrypt.GenerateFromPassword([]byte(req.Password), bcrypt.DefaultCost)
    if err != nil {
        return nil, err
    }

    // 创建用户
    user := &model.User{
        Email:    req.Email,
        Password: string(hashedPassword),
        Nickname: req.Nickname,
        Status:   1,
    }

    if err := s.repo.Create(ctx, user); err != nil {
        return nil, err
    }

    log.Log.Infof("用户创建成功：%d", user.ID)
    return user, nil
}

// Update 更新用户
func (s *UserService) Update(ctx context.Context, id int64, req *model.UpdateUserRequest) (*model.User, error) {
    user, err := s.repo.GetByID(ctx, id)
    if err != nil {
        return nil, err
    }

    // 更新字段
    if req.Nickname != "" {
        user.Nickname = req.Nickname
    }
    if req.Avatar != "" {
        user.Avatar = req.Avatar
    }

    if err := s.repo.Update(ctx, user); err != nil {
        return nil, err
    }

    // 删除缓存
    s.cache.Del(fmt.Sprintf("user:%d", id))
    log.Log.Infof("用户更新成功：%d", id)
    return user, nil
}

// Delete 删除用户
func (s *UserService) Delete(ctx context.Context, id int64) error {
    if err := s.repo.Delete(ctx, id); err != nil {
        return err
    }

    // 删除缓存
    s.cache.Del(fmt.Sprintf("user:%d", id))
    log.Log.Infof("用户删除成功：%d", id)
    return nil
}

// Login 用户登录
func (s *UserService) Login(ctx context.Context, email, password string) (*model.LoginResponse, error) {
    // 获取用户
    user, err := s.repo.GetByEmail(ctx, email)
    if err != nil {
        return nil, errors.New("用户不存在")
    }

    // 验证密码
    if err := bcrypt.CompareHashAndPassword([]byte(user.Password), []byte(password)); err != nil {
        return nil, errors.New("密码错误")
    }

    // 检查状态
    if user.Status != 1 {
        return nil, errors.New("账号已被禁用")
    }

    // 生成 Token
    token, expiresAt, err := jwt.GenerateToken(jwt.Claims{
        UserID: user.ID,
        Email:  user.Email,
    })
    if err != nil {
        return nil, err
    }

    log.Log.Infof("用户登录成功：%s", email)
    return &model.LoginResponse{
        Token:     token,
        ExpiresAt: expiresAt,
        User:      user,
    }, nil
}

// Register 用户注册
func (s *UserService) Register(ctx context.Context, req *model.CreateUserRequest) (*model.LoginResponse, error) {
    // 创建用户
    user, err := s.Create(ctx, req)
    if err != nil {
        return nil, err
    }

    // 自动登录
    token, expiresAt, err := jwt.GenerateToken(jwt.Claims{
        UserID: user.ID,
        Email:  user.Email,
    })
    if err != nil {
        return nil, err
    }

    return &model.LoginResponse{
        Token:     token,
        ExpiresAt: expiresAt,
        User:      user,
    }, nil
}
```

### 3.4 HTTP 处理器

```go
package handler

import (
    "errors"
    "net/http"
    "strconv"

    "user-service/internal/model"
    "user-service/internal/service"

    "vigo/framework/mvc"
    "vigo/framework/validate"
)

// UserHandler 用户处理器
type UserHandler struct {
    service *service.UserService
}

// NewUserHandler 创建用户处理器
func NewUserHandler(service *service.UserService) *UserHandler {
    return &UserHandler{
        service: service,
    }
}

// List 用户列表
func (h *UserHandler) List(c *mvc.Context) {
    page, _ := strconv.Atoi(c.Query("page", "1"))
    pageSize, _ := strconv.Atoi(c.Query("page_size", "10"))

    users, total, err := h.service.GetList(c.Request.Context(), page, pageSize)
    if err != nil {
        c.Json(http.StatusInternalServerError, errorResponse(err))
        return
    }

    c.Json(http.StatusOK, successResponse(map[string]interface{}{
        "list":  users,
        "total": total,
    }))
}

// Detail 用户详情
func (h *UserHandler) Detail(c *mvc.Context) {
    id, _ := strconv.ParseInt(c.Param("id"), 10, 64)

    user, err := h.service.GetByID(c.Request.Context(), id)
    if err != nil {
        c.Json(http.StatusNotFound, errorResponse(err))
        return
    }

    c.Json(http.StatusOK, successResponse(user))
}

// Create 创建用户
func (h *UserHandler) Create(c *mvc.Context) {
    var req model.CreateUserRequest
    if err := c.Bind(&req); err != nil {
        c.Json(http.StatusBadRequest, errorResponse(validate.Errors(err)))
        return
    }

    if err := validate.Struct(&req); err != nil {
```

```go
    }

// GetUser 获取用户信息
func (c *UserServiceClient) GetUser(ctx context.Context, userID int64) (*User, error) {

    ID       int64  `json:"id"`
    Email    string `json:"email"`
    Nickname string `json:"nickname"`

// NewOrderService 创建订单服务
func NewOrderService(repo *repository.OrderRepository, userService *UserServiceClient) *OrderService {
    return &OrderService{
        repo:        repo,
        userService: userService,
    }
}

// Create 创建订单
func (s *OrderService) Create(ctx context.Context, req *model.CreateOrderRequest) (*model.Order, error) {
    // 1. 验证用户是否存在（调用用户服务）
    if _, err := s.userService.GetUser(ctx, req.UserID); err != nil {
        return nil, fmt.Errorf("用户不存在：%w", err)
    }

    // 2. 生成订单号
    orderNo := generateOrderNo()

    // 3. 创建订单
    order := &model.Order{
        OrderNo:     orderNo,
        UserID:      req.UserID,
        TotalAmount: calculateTotalAmount(req.Items),
        Status:      0,
        Remark:      req.Remark,
    }

    // 4. 创建订单项
    for _, item := range req.Items {
        order.Items = append(order.Items, model.OrderItem{
            ProductID: item.ProductID,
            Quantity:  item.Quantity,
            // Price: 需要从商品服务获取
        })
    }

    // 5. 保存到数据库
    if err := s.repo.Create(ctx, order); err != nil {
        return nil, err
    }

    log.Log.Infof("订单创建成功：%s", order.OrderNo)
    return order, nil
}

// generateOrderNo 生成订单号
func generateOrderNo() string {
    return time.Now().Format("20060102150405") + fmt.Sprintf("%06d", rand.Intn(1000000))
}

// calculateTotalAmount 计算总金额
func calculateTotalAmount(items []struct {
    ProductID int64 `json:"product_id"`
    Quantity  int   `json:"quantity"`
}) int64 {
    var total int64
    for _, item := range items {
        // 这里需要从商品服务获取价格
        total += 1000 * int64(item.Quantity) // 示例：每个商品 10 元
    }
    return total
}
```

```go
func initDiscovery(cfg *config.Config) error {
    // 初始化服务发现客户端
    if err := discovery.Init(discovery.Config{
        Type:        "nacos",
        ServerAddr:  cfg.Discovery.ServerAddr,
        NamespaceID: cfg.Discovery.NamespaceID,

    }); err != nil {

    return discovery.Register(discovery.Instance{
        Name:      cfg.App.Name,
        IP:        getLocalIP(),
        Port:      cfg.App.Port,
        Weight:    cfg.Discovery.Weight,
        Healthy:   true,
        Enable:    true,
    })
}

// getLocalIP 获取本地 IP
func getLocalIP() string {
    addrs, err := net.InterfaceAddrs()
    if err != nil {
        return "127.0.0.1"
    }

    for _, addr := range addrs {
        if ipnet, ok := addr.(*net.IPNet); ok && !ipnet.IP.IsLoopback() {
            if ipnet.IP.To4() != nil {
                return ipnet.IP.String()
            }
        }
    }
    return "127.0.0.1"
}
```

```bash
go run cmd/migrate/main.go up
```

### 8.3 迁移管理

```bash
# 查看迁移状态
go run cmd/migrate/main.go status

# 回滚最后一个迁移
go run cmd/migrate/main.go down

# 回滚所有迁移
go run cmd/migrate/main.go reset

# 刷新迁移（先 down 再 up）
go run cmd/migrate/main.go refresh
```

## 10. 常见问题

### 10.1 服务间循环依赖

**问题**：用户服务依赖订单服务，订单服务又依赖用户服务

**解决方案**：

1. 提取公共模块到独立服务
2. 使用事件驱动架构
3. 数据冗余（在订单服务中冗余用户基本信息）

### 10.2 分布式事务

**问题**：跨服务操作如何保证数据一致性

**解决方案**：

1. **最终一致性**：使用消息队列
2. **Saga 模式**：长事务拆分为多个本地事务
3. **TCC 模式**：Try-Confirm-Cancel

### 10.3 服务雪崩

**问题**：一个服务故障导致整个系统崩溃

**解决方案**：

1. **超时控制**：设置合理的超时时间
2. **熔断器**：使用 Hystrix 或 Sentinel
3. **限流降级**：限制请求量，保护核心服务
4. **隔离**：线程池隔离、信号量隔离

### 10.4 日志收集

**问题**：分布式系统日志分散

**解决方案**：

1. **ELK Stack**：Elasticsearch + Logstash + Kibana
2. **Loki**：轻量级日志系统
3. **统一日志格式**：JSON 格式，包含 trace_id

### 10.5 链路追踪

**问题**：请求在多个服务间流转，难以定位问题

**解决方案**：

1. **Jaeger**：开源分布式追踪系统
2. **Zipkin**：Twitter 开源的追踪系统
3. **SkyWalking**：国产 APM 系统
