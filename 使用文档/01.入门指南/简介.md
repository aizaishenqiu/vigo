# 简介

**Vigo** 是一个免费开源的，快速、简单的面向对象的轻量级 Go 语言 Web 开发框架。

它是为了敏捷 WEB 应用开发和简化企业应用开发而诞生的。Vigo 秉承简洁实用的设计原则，在保持出色的性能和至简代码的同时，更注重易用性。

## 设计理念

框架深受主流 Web 框架开发模式的影响，致力于在 Go 语言生态中提供简洁、优雅的开发体验，降低开发者的学习和迁移成本。

## 核心特性

- **极简开发**：借鉴主流框架的优秀设计，通过链式操作和 Active Record 模式极大提升开发效率。
- **内置安全**：自动加载 XSS 过滤、DoS 防护、异常恢复等中间件，为应用保驾护航。
- **高性能**：基于原生 Go 语言构建，核心模块轻量且高效，满足高并发场景需求。
- **读写分离**：内置数据库读写分离支持，支持主从配置与负载均衡。
- **模块化设计**：容器化管理与门面模式，让业务逻辑与底层实现深度解耦。
- **全栈监控**：自带实时运行状态监控界面，Redis、RabbitMQ 等服务状态一目了然。

## 适用场景

- 企业级 SaaS 系统
- API 接口服务
- 中小型 Web 应用
- 微服务模块

## 开源协议

遵循 Apache 2.0 开源许可协议发布。
