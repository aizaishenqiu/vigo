# 🚀 快速开始

## 环境要求

- **Go**: 1.18+
- **Git**: 最新版
- **Air** (可选，推荐): 用于开发热重载

## 安装与启动

### 1. 下载项目

```bash
git clone https://gitee.com/yjk100_admin/vigo.git
cd vigo
```

### 2. 安装依赖

```bash
go mod tidy
```

### 3. 运行项目 (开发模式)

推荐使用 `air` 进行热重载开发：

```bash
# 如果已安装 air
air

# 或者使用 go run 运行 air
go run github.com/air-verse/air@latest
```

启动成功后，终端会显示：

```
Starting vigo v3.0.0 (Mode: dev)...
Listening on :8080
```

访问浏览器：`http://localhost:8080`

## 📁 目录结构

```
vigo/
├── app/
│   ├── controller/      # 控制器 (业务逻辑)
│   ├── model/           # 模型 (数据操作)
│   ├── service/         # 服务层 (复杂业务)
│   └── view/            # 视图文件 (HTML模板)
├── config/              # 全局配置加载逻辑
├── framework/           # 框架核心库
│   ├── db/              # 数据库组件
│   ├── mvc/             # MVC 核心 (Context, Router, Controller)
│   └── ...
├── route/               # 路由定义
├── docs/                # Swagger API 文档
├── config.yaml          # 核心配置文件
├── config.local.yaml    # 本地覆盖配置 (不进入版本库)
├── .air.toml            # Air 热重载配置
├── main.go              # 项目入口
└── go.mod               # 依赖管理
```
