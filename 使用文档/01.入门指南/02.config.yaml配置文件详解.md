# config.yaml 配置文件完整说明

## 概述

`config.yaml` 是 Vigo 框架的主配置文件，采用 YAML 格式，支持环境变量占位符。

### 环境变量语法

```yaml
# 基本格式
${ENV_VAR}

# 带默认值格式
${ENV_VAR:default_value}
```

**示例**：
```yaml
database:
  host: ${DB_HOST}           # 从 DB_HOST 环境变量读取
  port: ${DB_PORT:3306}      # 从 DB_PORT 读取，未设置则使用 3306
  name: ${DB_NAME:myapp}     # 从 DB_NAME 读取，未设置则使用 myapp
```
