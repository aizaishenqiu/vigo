# 定时任务调度器使用指南

## 概述

Vigo 框架提供基于 Cron 的定时任务调度器，支持秒级任务、任务管理、状态监控等功能。

## 功能特性

- ✅ 支持秒级 Cron 表达式
- ✅ 任务启用/禁用
- ✅ 立即执行任务
- ✅ 任务状态监控
- ✅ 并发控制
- ✅ 任务元数据管理

## 快速开始

### 1. 基本使用

```go
package main

import (
    "fmt"
    "time"
    "vigo/framework/scheduler"
)

func main() {
    // 创建调度器
    sched := scheduler.NewScheduler(&scheduler.SchedulerOptions{
        Location: time.Local,
    })

    // 添加定时任务（每天凌晨 2 点执行）
    sched.AddTask("daily-report", "0 0 2 * * *", func() {
        fmt.Println("Generating daily report...")
        generateDailyReport()
    })

    // 添加定时任务（每小时执行）
    sched.AddTask("hourly-cleanup", "0 0 * * * *", func() {
        fmt.Println("Cleaning up...")
        cleanupTempData()
    })

    // 启动调度器
    sched.Start()

    // 保持运行
    select {}
}
```

## Cron 表达式

### 格式

```
秒 分 时 日 月 周
```

### 示例

| 表达式 | 说明 |
|--------|------|
| `0 0 2 * * *` | 每天凌晨 2 点 |
| `0 30 8 * * 1-5` | 工作日早上 8:30 |
| `0 */5 * * * *` | 每 5 分钟 |
| `0 0 */2 * * *` | 每 2 小时 |
| `0 0 0 1 * *` | 每月 1 号凌晨 |
| `0 0 0 * * 0` | 每周日凌晨 |

### 特殊字符

- `*` - 每个值（如 `*` 在小时字段表示每小时）
- `,` - 分隔多个值（如 `1,3,5`）
- `-` - 范围（如 `1-5`）
- `/` - 步长（如 `*/5` 表示每 5 个单位）

## 配置选项

### SchedulerOptions

```go
type SchedulerOptions struct {
    // 时区
    Location *time.Location `yaml:"location"`
    
    // 是否立即执行第一次
    RunImmediately bool `yaml:"run_immediately"`
}
```

### Task

```go
type Task struct {
    Name        string                 `json:"name"`
    Spec        string                 `json:"spec"`
    Description string                 `json:"description"`
    Enabled     bool                   `json:"enabled"`
    Running     bool                   `json:"running"`
    LastRun     time.Time              `json:"last_run"`
    NextRun     time.Time              `json:"next_run"`
    TotalRuns   int                    `json:"total_runs"`
    Metadata    map[string]interface{} `json:"metadata"`
}
```

## 基本操作

### 1. 添加任务

```go
// 简单任务
sched.AddTask("task-name", "0 * * * * *", func() {
    // 任务逻辑
})

// 带描述的任务
sched.AddTaskWithDesc("task-name", "0 * * * * *", "任务描述", func() {
    // 任务逻辑
})

// 带 Context 的任务
sched.AddTaskWithCtx("task-name", "0 * * * * *", func(ctx context.Context) {
    // 任务逻辑（支持取消）
    doWork(ctx)
})
```

### 2. 移除任务

```go
err := sched.RemoveTask("task-name")
```

### 3. 启用/禁用任务

```go
// 启用任务
err := sched.EnableTask("task-name")

// 禁用任务
err := sched.DisableTask("task-name")
```

### 4. 立即执行

```go
// 立即执行任务
err := sched.RunTaskNow("task-name")
```

### 5. 获取任务信息

```go
// 获取任务
task, err := sched.GetTask("task-name")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Task: %+v\n", task)

// 获取下次执行时间
nextRun, err := sched.GetNextRun("task-name")
fmt.Printf("Next run: %v\n", nextRun)

// 获取上次执行时间
lastRun, err := sched.GetPrevRun("task-name")
fmt.Printf("Last run: %v\n", lastRun)
```

### 6. 任务列表

```go
// 获取所有任务
tasks := sched.ListTasks()
for _, task := range tasks {
    fmt.Printf("%s: %s (enabled: %v, next: %v)\n",
        task.Name, task.Spec, task.Enabled, task.NextRun)
}

// 获取所有任务状态
statuses := sched.ListTaskStatuses()
for _, status := range statuses {
    fmt.Printf("%s: runs=%d, last=%v, next=%v\n",
        status.Name, status.TotalRuns, status.LastRun, status.NextRun)
}
```

### 7. 统计信息

```go
stats := sched.GetStats()
fmt.Printf("Total tasks: %d\n", stats.TotalTasks)
fmt.Printf("Enabled tasks: %d\n", stats.EnabledTasks)
fmt.Printf("Running tasks: %d\n", stats.RunningTasks)
fmt.Printf("Scheduler running: %v\n", stats.IsRunning)
```

### 8. 元数据管理

```go
// 设置元数据
err := sched.SetMetadata("task-name", "retry_count", 3)
err := sched.SetMetadata("task-name", "timeout", 30*time.Second)

// 获取元数据
value, err := sched.GetMetadata("task-name", "retry_count")
retryCount := value.(int)
```

## 完整示例

### 日报生成任务

```go
package tasks

import (
    "context"
    "fmt"
    "log"
    "time"
    "vigo/framework/scheduler"
)

type ReportTask struct {
    scheduler *scheduler.Scheduler
}

func NewReportTask() *ReportTask {
    return &ReportTask{
        scheduler: scheduler.NewScheduler(&scheduler.SchedulerOptions{
            Location: time.Local,
        }),
    }
}

// 初始化任务
func (r *ReportTask) Init() error {
    // 每天凌晨 2 点生成日报
    err := r.scheduler.AddTaskWithCtx("daily-report", "0 0 2 * * *", func(ctx context.Context) {
        log.Println("Starting daily report generation...")
        
        if err := r.generateReport(ctx); err != nil {
            log.Printf("Daily report failed: %v", err)
            return
        }
        
        log.Println("Daily report completed")
    })
    
    if err != nil {
        return err
    }

    // 每周一早上 8 点生成周报
    err = r.scheduler.AddTaskWithCtx("weekly-report", "0 0 8 * * 1", func(ctx context.Context) {
        log.Println("Starting weekly report generation...")
        
        if err := r.generateWeeklyReport(ctx); err != nil {
            log.Printf("Weekly report failed: %v", err)
            return
        }
        
        log.Println("Weekly report completed")
    })
    
    return err
}

// 生成日报
func (r *ReportTask) generateReport(ctx context.Context) error {
    // 检查是否超时
    ctx, cancel := context.WithTimeout(ctx, 30*time.Minute)
    defer cancel()

    // 1. 获取统计数据
    stats, err := getDailyStats(ctx)
    if err != nil {
        return err
    }

    // 2. 生成报告
    report := &Report{
        Date:  time.Now().AddDate(0, 0, -1),
        Stats: stats,
    }

    // 3. 发送报告
    return sendReport(report)
}

// 生成周报
func (r *ReportTask) generateWeeklyReport(ctx context.Context) error {
    // 类似日报逻辑
    return nil
}

// 启动调度器
func (r *ReportTask) Start() {
    r.scheduler.Start()
    log.Println("Report scheduler started")
}

// 停止调度器
func (r *ReportTask) Stop() {
    r.scheduler.Stop()
    log.Println("Report scheduler stopped")
}

// 手动触发日报

    return r.scheduler.RunTaskNow("daily-report")

    return r.scheduler.GetStats()
```

## 最佳实践

### 1. 错误处理

```go
sched.AddTaskWithCtx("task", "0 * * * * *", func(ctx context.Context) {
    defer func() {
        if r := recover(); r != nil {
            log.Printf("Task panic: %v", r)
            // 发送告警
            sendAlert("Task panic", r)
        }
    }()

    if err := doWork(ctx); err != nil {
        log.Printf("Task error: %v", err)
        // 记录错误
        recordError(err)
    }
})
```

### 2. 超时控制

```go
sched.AddTaskWithCtx("task", "0 * * * * *", func(ctx context.Context) {
    // 设置超时
    ctx, cancel := context.WithTimeout(ctx, 5*time.Minute)
    defer cancel()

    // 执行任务
    doWork(ctx)
})
```

### 3. 任务重试

```go
sched.AddTaskWithCtx("task", "0 * * * * *", func(ctx context.Context) {
    maxRetries := 3
    var err error

    for i := 0; i < maxRetries; i++ {
        if err = doWork(ctx); err == nil {
            return
        }
        
        log.Printf("Task retry %d: %v", i+1, err)
        time.Sleep(time.Second * time.Duration(i))
    }

    // 所有重试失败
    sendAlert("Task failed after retries", err)
})
```

### 4. 任务监控

```go
// 定期检查任务状态
go func() {
    ticker := time.NewTicker(time.Minute)
    defer ticker.Stop()

    for range ticker.C {
        stats := sched.GetStats()
        
        // 检查是否有任务长时间未运行
        for _, task := range sched.ListTasks() {
            if task.Enabled && time.Since(task.LastRun) > time.Hour {
                sendAlert("Task not running", task.Name)
            }
        }
        
        // 记录统计信息
        metrics.Set("scheduler.total_tasks", stats.TotalTasks)
        metrics.Set("scheduler.enabled_tasks", stats.EnabledTasks)
        metrics.Set("scheduler.running_tasks", stats.RunningTasks)
    }
}()
```

### 5. 优雅停机

```go
func gracefulShutdown(sched *scheduler.Scheduler) {
    // 停止调度器
    sched.Stop()
    
    // 等待正在运行的任务完成
    time.Sleep(10 * time.Second)
    
    log.Println("Scheduler shutdown complete")
}
```

## 故障排查

### 1. 任务不执行

```go
// 检查调度器是否启动
stats := sched.GetStats()
if !stats.IsRunning {
    log.Fatal("Scheduler not running")
}

// 检查任务是否启用
task, _ := sched.GetTask("task-name")
if !task.Enabled {
    log.Fatal("Task not enabled")
}

// 检查 Cron 表达式
nextRun, _ := sched.GetNextRun("task-name")
log.Printf("Next run: %v", nextRun)
```

### 2. 任务重复执行

```go
// 检查是否有多个调度器实例
// 使用分布式锁防止重复执行
if lock.Acquire("task:lock:task-name") {
    defer lock.Release("task:lock:task-name")
    doWork()
}
```

### 3. 任务执行慢

```go
// 记录任务执行时间
start := time.Now()
doWork()
duration := time.Since(start)

if duration > time.Minute {
    log.Printf("Task slow: %v", duration)
}
```
