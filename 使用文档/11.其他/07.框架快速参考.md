## 快速开始

### 1. 创建项目

```bash
# 使用 vigoctl 创建新项目
vigoctl new project myapp

cd myapp
go mod tidy
```

### 2. 配置数据库

```go
import "vigo/framework/db"

// 创建数据库连接
db, err := db.NewDB(&db.DBConfig{
    Driver: "mysql",
    DSN:    "user:pass@tcp(127.0.0.1:3306)/dbname?charset=utf8mb4",
    MaxIdleConns: 10,
    MaxOpenConns: 100,
})

// 启用连接池监控
db.EnablePoolMonitor(5 * time.Second)
```

### 3. 配置 Redis

```go
import "vigo/framework/cache"

// 创建 Redis 客户端
client, err := cache.CreateRedisClient(
    "127.0.0.1:6379",
    "",
    0,
    &cache.RedisCacheOptions{
        Prefix:     "app:",
        DefaultTTL: 24 * time.Hour,
        PoolSize:   100,
    },
)

// 创建缓存适配器
redisCache := cache.NewRedisCacheAdapter(client, &cache.RedisCacheOptions{
    Prefix: "app:",
})
```

### 4. 配置 Nacos

```go
import "vigo/framework/config"

// 创建 Nacos 配置
nacosCfg, err := config.NewNacosConfig(&config.NacosConfigOptions{
    ServerAddr:  "127.0.0.1:8848",
    NamespaceId: "public",
    DataId:      "app.yaml",
    Group:       "DEFAULT_GROUP",
    Format:      "yaml",
})

// 加载配置
nacosCfg.Load(ctx)

// 监听配置变化
go nacosCfg.Watch(ctx)

// 获取配置
dbDSN, _ := nacosCfg.Get("database.dsn")
```

### 5. 服务注册

```go
import "vigo/framework/discovery"

// 创建服务发现
disc, err := discovery.NewNacosDiscovery(&discovery.NacosDiscoveryOptions{
    ServerAddr:  "127.0.0.1:8848",
    NamespaceId: "public",
})

// 注册服务
disc.Register(ctx, "my-service", &discovery.Instance{
    IP:   "192.168.1.100",
    Port: 8080,
    Weight: 1.0,
})
```

### 6. 配置限流

```go
import "vigo/framework/limiter"

// 创建分布式限流器
limiter := limiter.NewDistributedLimiter(&limiter.DistributedLimiterOptions{
    Client:   redisClient,
    Key:      "api:limit:user:1",
    Rate:     100,
    Burst:    200,
    Interval: time.Second,
})

// 使用限流器
if limiter.Allow() {
    // 处理请求
} else {
    // 拒绝请求
}
```

### 7. 创建定时任务

```go
import "vigo/framework/scheduler"

// 创建调度器
sched := scheduler.NewScheduler(&scheduler.SchedulerOptions{
    Location: time.Local,
})

// 添加定时任务
sched.AddTask("daily-report", "0 0 2 * * *", func() {
    generateReport()
})

// 启动调度器
sched.Start()
```

## 核心 API

### 路由与控制器

```go
// 定义路由
r := vigo.Default()
r.GET("/user/:id", UserController.Get)
r.POST("/user", UserController.Create)
r.PUT("/user/:id", UserController.Update)
r.DELETE("/user/:id", UserController.Delete)

// 控制器
type UserController struct{}

func (c *UserController) Get(ctx *vigo.Context) {
    id := ctx.Param("id")
    user, _ := model.User.Find(id)
    ctx.JSON(200, user)
}
```

### 数据库操作

```go
// 查询
user, _ := db.Table("users").Where("id", 1).First()
users, _ := db.Table("users").WhereIn("status", []int{1, 2}).Get()

// 写入
db.Table("users").Insert(map[string]interface{}{
    "name": "John",
    "age":  25,
})

// 更新
db.Table("users").Where("id", 1).Update(map[string]interface{}{
    "name": "Jane",
})

// 删除
db.Table("users").Where("id", 1).Delete()
```

### 缓存操作

```go
// 设置缓存
redisCache.Set("key", value, time.Hour)

// 获取缓存
value, _ := redisCache.Get("key")

// 删除缓存
redisCache.Delete("key")

// 批量设置
redisCache.SetMulti(map[string]interface{}{
    "key1": value1,
    "key2": value2,
}, time.Hour)

// 批量获取
values, _ := redisCache.GetMulti([]string{"key1", "key2"})
```

### 中间件

```go
// 全局中间件
r.Use(middleware.Logger())
r.Use(middleware.Recovery())
r.Use(middleware.CORS())

// 分组中间件
api := r.Group("/api", middleware.Auth())
{
    api.GET("/user", UserController.List)
}

// 自定义中间件
func MyMiddleware() vigo.HandlerFunc {
    return func(ctx *vigo.Context) {
        // 前置处理
        ctx.Next()
        // 后置处理
    }
}
```

## 故障排查

### 常见问题

**Q: 数据库连接失败？**
```bash
# 检查数据库状态
mysql -u root -p

# 检查网络连接
telnet 127.0.0.1 3306
```

**Q: Redis 连接失败？**
```bash
# 检查 Redis 状态
redis-cli ping

docker logs redis
```

**Q: Nacos 连接失败？**
```bash
# 检查 Nacos 状态
curl http://127.0.0.1:8848/nacos/

tail -f nacos/logs/nacos.log

# 清理模块缓存
go clean -modcache

go mod tidy
```

pool := grpcpool.NewPool(&grpcpool.Options{

// 多级缓存
cache := multilevel.NewMultiLevelCache(

// 启用 HTTP/2
&http2.ServerOptions{
