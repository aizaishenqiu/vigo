# 📦 缓存管理 (Cache)

框架提供了统一的缓存接口，支持内存（Memory）、文件（File）和多级缓存三种实现。

## 初始化

```go
import "vigo/framework/cache"

var Cache cache.Cache

func init() {
    // 使用内存缓存（推荐用于开发环境）
    Cache = cache.NewMemoryCache(1000)

    // 或者使用文件缓存
    // Cache = cache.NewFileCache("runtime/cache")

    // 或者使用多级缓存（内存 + Redis）
    // Cache = cache.NewMultiLevelCache(1000, 30*time.Minute)
}
```

## 使用方法

### 设置缓存

```go
// key, value, ttl (过期时间)
Cache.Set("user:1", userData, 10*time.Minute)
```

### 获取缓存

```go
val := Cache.Get("user:1")
if val == nil {
    // 缓存不存在或已过期
} else {
    // 类型断言
    user := val.(UserStruct)
}
```

### 判断存在

```go
if Cache.Has("user:1") {
    // ...
}
```

### 删除缓存

```go
Cache.Delete("user:1")
```

### 清空缓存

```go
Cache.Flush()
```

## 多级缓存

```go
// 创建多级缓存
mc := cache.NewMultiLevelCache(1000, 30*time.Minute)

// 使用 GetWithProtection 防止缓存击穿
data, err := mc.GetWithProtection("user:1", func() (interface{}, error) {
    // 从数据库查询
    user, err := db.NewQuery(db.GetDB()).
        Table("users").
        Where("id", 1).
        Find()
    return user, err
})
```
