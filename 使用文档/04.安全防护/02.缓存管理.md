# 📦 缓存管理 (Cache)

框架提供了统一的缓存接口，支持内存（Memory）和文件（File）两种驱动。

## 初始化

```go
import "vigo/framework/cache"

var Cache cache.Cache

func init() {
    // 使用内存缓存
    Cache = cache.NewMemoryCache()
    
    // 或者使用文件缓存
    // Cache = cache.NewFileCache("runtime/cache")
}
```

## 使用方法

### 设置缓存
```go
// key, value, ttl (过期时间)
Cache.Set("user:1", userData, 10*time.Minute)
```

### 获取缓存
```go
val := Cache.Get("user:1")
if val == nil {
    // 缓存不存在或已过期
} else {
    // 类型断言
    user := val.(UserStruct)
}
```

### 判断存在
```go
if Cache.Has("user:1") {
    // ...
}
```

### 删除缓存
```go
Cache.Delete("user:1")
```
