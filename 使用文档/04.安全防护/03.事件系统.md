# 📡 事件系统 (Event)

框架提供了简单的事件监听与分发机制，用于解耦业务逻辑。

## 注册监听 (Listen)

通常在应用启动时（如 `app/event/listener.go`）注册。

```go
import (
    "vigo/framework/event"
    "fmt"
)

func InitListeners() {
    // 监听 "user.login" 事件
    event.Listen("user.login", func(data interface{}) {
        user := data.(map[string]string)
        fmt.Printf("User %s logged in, sending notification...\n", user["username"])
    })
}
```

## 触发事件 (Trigger)

在业务逻辑中触发事件。注意：事件监听器是**异步执行**的。

```go
func (c *AuthController) Login(ctx *mvc.Context) {
    // ... 登录成功逻辑 ...
    
    // 触发登录事件
    event.Trigger("user.login", map[string]string{
        "id": "1", 
        "username": "admin",
    })
}
```
