# 📋 日志处理 (Log)

框架内置了基础的日志记录功能，支持文件和控制台双重输出。

## 初始化

在应用启动时（如 `main.go` 或 `config.Init` 中）进行初始化：

```go
import "vigo/framework/log"

func main() {
    // 初始化日志，指定日志存储目录
    log.Init("runtime/log")
}
```

## 使用方法

支持 `Debug`, `Info`, `Error` 三种级别。

```go
import "vigo/framework/log"

func (c *UserController) Index(ctx *mvc.Context) {
    // 记录普通信息
    log.Log.Info("User accessed index page")
    
    // 记录调试信息
    log.Log.Debug("Processing request params: " + c.Input("id"))
    
    // 记录错误信息
    if err != nil {
        log.Log.Error("Database connection failed: " + err.Error())
    }
}
```

## 日志格式

日志文件按日期和级别自动分割，例如：
`runtime/log/2026-02-15-INFO.log`

内容格式：
`[2026-02-15 12:00:00] [INFO] User accessed index page`
