# 缓存增强使用指南

Vigo 框架提供强大的缓存增强功能，包括标签管理、缓存依赖、多级缓存、Remember 模式等高级特性。

## 一、快速开始

### 1.1 基本使用

```go
package main

import (
    "time"
    "vigo/framework/cache"
)

func main() {
    // 创建缓存
    c := cache.NewMemoryCache()
    
    // 设置缓存
    c.Set("user:1", userData, 1*time.Hour)
    
    // 获取缓存
    value := c.Get("user:1")
    
    // 删除缓存
    c.Delete("user:1")
}
```

## 二、标签管理

### 2.1 使用标签

```go
// 创建带标签的缓存
taggedCache := cache.NewTaggedCache(c)

// 设置带标签的缓存
taggedCache.Tag("user", "vip").Set("user:1", userData, 1*time.Hour)
taggedCache.Tag("user", "vip").Set("user:2", userData2, 1*time.Hour)

// 清空指定标签的所有缓存
taggedCache.ClearTags("user")  // 清空所有 user 标签的缓存
taggedCache.ClearTags("vip")   // 清空所有 vip 标签的缓存

// 清空多个标签
taggedCache.ClearTags("user", "vip")
```

**应用场景**:
- 批量清除用户相关缓存
- 按类型清除缓存
- 版本化管理缓存

---

### 2.2 标签组合

```go
// 多个标签组合
taggedCache.Tag("article", "category:1", "author:5").
    Set("article:100", articleData, 2*time.Hour)

// 清除某分类下的所有文章缓存
taggedCache.ClearTags("category:1")

// 清除某作者的所有文章缓存
taggedCache.ClearTags("author:5")
```

## 三、缓存依赖

### 3.1 基本依赖

```go
// 创建缓存依赖
dep := cache.NewCacheDependency("config:version")

// 创建带依赖的缓存
cacheWithDep := cache.NewCacheWithDependency(c, dep)

// 设置缓存（当 config:version 被删除时，此缓存也失效）
cacheWithDep.Set("user:settings", settings, 24*time.Hour)

// 删除依赖键，缓存自动失效
c.Delete("config:version")  // user:settings 也会失效
```

---

### 3.2 多依赖

```go
// 多个依赖
dep := cache.NewCacheDependency("config:version", "user:1:role")

cacheWithDep := cache.NewCacheWithDependency(c, dep)
cacheWithDep.Set("user:1:permissions", permissions, 24*time.Hour)

// 任意一个依赖键被删除，缓存都失效
c.Delete("config:version")      // permissions 失效
c.Delete("user:1:role")         // permissions 也失效
```

**应用场景**:
- 配置变更时清除相关缓存
- 用户角色变更时清除权限缓存
- 数据更新时清除查询缓存

## 四、Remember 模式

### 4.1 基本用法

```go
// 缓存不存在时执行闭包并缓存结果
value, err := c.Remember("user:1", 1*time.Hour, func() interface{} {
    // 从数据库查询
    return db.Query("SELECT * FROM users WHERE id = ?", 1)
})
```

**特点**:
- 自动判断缓存是否存在
- 缓存不存在时执行闭包
- 自动缓存闭包返回值

---

### 4.2 带错误处理

```go
value, err := c.Remember("user:1", 1*time.Hour, func() (interface{}, error) {
    user, err := db.Query("SELECT * FROM users WHERE id = ?", 1)
    if err != nil {
        return nil, err
    }
    return user, nil
})
```

---

### 4.3 RememberForever

```go
// 永久缓存（直到手动删除）
value := c.RememberForever("site:config", func() interface{} {
    return db.Query("SELECT * FROM site_config LIMIT 1")
})
```

## 五、多级缓存

### 5.1 创建多级缓存

```go
// 内存缓存（L1）
l1 := cache.NewMemoryCache()

// Redis 缓存（L2）
l2 := cache.NewRedisCache(redisClient)

// 创建多级缓存
multiCache := cache.NewMultiLevelCache(l1, l2)

// 使用
multiCache.Set("user:1", userData, 1*time.Hour)
value := multiCache.Get("user:1")  // 优先从 L1 获取，L1 没有从 L2 获取
```

**特点**:
- L1 命中：极快（内存访问）
- L1 未命中：从 L2 获取并写入 L1
- 自动同步：L2 更新时通知 L1 失效

---

### 5.2 配置示例

```go
func NewCache() *cache.MultiLevelCache {
    // L1: 内存缓存，100MB
    l1 := cache.NewMemoryCacheWithSize(100 * 1024 * 1024)
    
    // L2: Redis 缓存
    redisClient := redis.NewClient(&redis.Options{
        Addr: "localhost:6379",
    })
    l2 := cache.NewRedisCache(redisClient)
    
    return cache.NewMultiLevelCache(l1, l2)
}
```

## 六、fail_delete 配置

### 6.1 启用 fail_delete

```yaml
# config.yaml
cache:
  fail_delete: true  # 缓存失效时自动删除
```

**行为**:
- `fail_delete: true`: 缓存失效时自动删除，下次访问重新生成
- `fail_delete: false`: 缓存失效后保留，直到被新值覆盖

---

### 6.2 使用场景

**推荐启用 fail_delete**:
- 数据一致性要求高
- 缓存空间有限
- 希望自动清理过期数据

**不启用 fail_delete**:
- 可以接受短暂过期数据
- 希望减少缓存击穿
- 缓存生成成本高

### 8.3 缓存穿透防护

```go
// 缓存空值防止穿透
func GetUser(id int) (*User, error) {
    key := fmt.Sprintf("user:%d", id)
    
    value, err := c.Remember(key, 1*time.Hour, func() interface{} {
        user, err := db.Query("SELECT * FROM users WHERE id = ?", id)
        if err == sql.ErrNoRows {
            // 缓存空值，避免频繁查询数据库
            return &User{}  // 空对象
        }
        return user
    })
    
    return value.(*User), err
}
```

## 九、完整示例

### 9.1 用户数据缓存

```go
package service

type UserService struct {
    cache *cache.TaggedCache
}

func NewUserService(c *cache.TaggedCache) *UserService {
    return &UserService{cache: c}
}

// GetUser 获取用户（带缓存）
func (s *UserService) GetUser(id int) (*User, error) {
    key := fmt.Sprintf("user:%d", id)
    
    value, err := s.cache.Remember(key, 1*time.Hour, func() interface{} {
        return db.QueryRow("SELECT * FROM users WHERE id = ?", id)
    })
    
    return value.(*User), err
}

// UpdateUser 更新用户（清除缓存）
func (s *UserService) UpdateUser(user *User) error {
    // 更新数据库
    _, err := db.Exec("UPDATE users SET ? WHERE id = ?", user, user.ID)
    if err != nil {
        return err
    }
    
    // 清除缓存
    s.cache.Delete(fmt.Sprintf("user:%d", user.ID))
    
    return nil
}

// GetUserArticles 获取用户文章（带标签）
func (s *UserService) GetUserArticles(userID int) ([]*Article, error) {
    key := fmt.Sprintf("user:%d:articles", userID)
    
    value, err := s.cache.Tag("user", "article").Remember(key, 30*time.Minute, func() interface{} {
        return db.Query("SELECT * FROM articles WHERE user_id = ?", userID)
    })
    
    return value.([]*Article), err
}

// ClearUserCache 清除用户所有缓存
func (s *UserService) ClearUserCache(userID int) {
    s.cache.ClearTags("user")  // 清除所有用户相关缓存
}
```

### 9.2 配置缓存

```go
package service

type ConfigService struct {
    cache *cache.CacheWithDependency
}

func NewConfigService(c cache.Cache) *ConfigService {
    // 创建配置版本依赖
    dep := cache.NewCacheDependency("config:version")
    cacheWithDep := cache.NewCacheWithDependency(c, dep)
    
    return &ConfigService{cache: cacheWithDep}
}

// GetConfig 获取配置
func (s *ConfigService) GetConfig(key string) (interface{}, error) {
    cacheKey := fmt.Sprintf("config:%s", key)
    
    value, err := s.cache.Remember(cacheKey, 24*time.Hour, func() interface{} {
        return db.QueryRow("SELECT value FROM config WHERE key = ?", key)
    })
    
    return value, err
}

// UpdateConfig 更新配置
func (s *ConfigService) UpdateConfig(key, value string) error {
    // 更新数据库
    _, err := db.Exec("UPDATE config SET value = ? WHERE key = ?", value, key)
    if err != nil {
        return err
    }
    
    // 更新配置版本，使所有配置缓存失效
    s.cache.GetCache().Increment("config:version")
    
    return nil
}
```

// L1: 本地内存缓存（热点数据）
