# RabbitMQ 消息队列

框架内置了完整的 RabbitMQ 客户端和 Web 管理界面，支持自动重连、队列管理、消息发布。

---

## 1. 配置

在 `config.yaml` 中配置：

```yaml
rabbitmq:
  enabled: true # 设为 true 启用
  host: 127.0.0.1
  port: 5672
  user: guest
  pass: guest
  vhost: /
  conn_timeout: 5 # 连接超时（秒）
  heartbeat: 10 # 心跳间隔（秒）
  reconnect_delay: 3 # 重连间隔（秒）
  max_retries: 3 # 最大重试次数（0=无限重试）

  # 管理界面配置
  admin:
    enabled: true # 启用管理界面
    username: "admin" # 管理界面用户名（留空则不启用认证）
    password: "your_password" # 管理界面密码（留空则不启用认证）
    port: 15672 # RabbitMQ Management 端口
```

```
║  RabbitMQ 服务: http://127.0.0.1:15672
╚════════════════════════════════════════════════════════════╝
```

管理界面功能：

- **队列管理** - 查看、清空、删除队列
- **交换机管理** - 查看交换机列表
- **连接管理** - 查看当前连接
- **消息统计** - 消息数、消费者数等

> 需要启用 RabbitMQ Management 插件才能获取实时数据。

## 3. 代码中使用

### 通过门面访问

```go
import "vigo/framework/facade"

mq := facade.RabbitMQ()
```

### 发布消息

```go
// 发到指定队列
err := mq.PublishToQueue("order.created", []byte(`{"order_id": 123}`))

// 通过交换机路由
err := mq.Publish("order.exchange", "order.created", []byte(`{"order_id": 123}`))
```

### 消费消息

```go
mq.Consume("order.created", func(body []byte) {
    fmt.Println("收到消息:", string(body))
    // 处理业务逻辑
})
```

### 队列管理

```go
mq.DeclareQueue("my_queue", true)     // 声明持久化队列
mq.DeleteQueue("my_queue")            // 删除队列
mq.PurgeQueue("my_queue")             // 清空队列

mq.DeclareExchange("my_ex", "topic", true)  // 声明交换机
mq.DeleteExchange("my_ex")                  // 删除交换机
```

### 状态检查

```go
mq.IsConnected()  // bool
mq.GetStatus()    // map: status, dsn, reconnecting
mq.GetOverview()  // RabbitMQ 版本、集群信息、消息统计
```

## 4. 自动重连

客户端内置自动重连机制：

- 连接断开后自动检测并重连
- 支持指数退避策略（最大 30 秒）
- 可配置最大重试次数

## 5. API 接口

| 方法   | 路径                                 | 说明                      |
| :----- | :----------------------------------- | :------------------------ |
| GET    | `/admin/rabbitmq`                    | 管理页面（需 Basic Auth） |
| GET    | `/admin/rabbitmq/queues`             | 队列列表                  |
| GET    | `/admin/rabbitmq/exchanges`          | 交换机列表                |
| GET    | `/admin/rabbitmq/connections`        | 连接列表                  |
| POST   | `/admin/rabbitmq/queue/{name}/purge` | 清空队列                  |
| DELETE | `/admin/rabbitmq/queue/{name}`       | 删除队列                  |
