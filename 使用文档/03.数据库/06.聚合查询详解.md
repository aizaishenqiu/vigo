# 聚合查询详解

## 简介
聚合查询用于对数据进行统计计算，包括计数、求和、平均值、最大值、最小值等。

## 聚合方法

### Count - 统计数量

```go
// 统计总数
count, err := db.Table("users").Count()
fmt.Printf("用户总数: %d\n", count)

// 条件统计
count, err = db.Table("users").
    Where("status", 1).
    Count()
fmt.Printf("激活用户: %d\n", count)

// 多条件统计
count, err = db.Table("users").
    Where("status", 1).
    Where("city", "北京").
    Count()
```

### Sum - 求和

```go
// 计算总和
total, err := db.Table("orders").
    Where("status", "completed").
    Sum("amount")
fmt.Printf("订单总金额: %.2f\n", total)

// 计算积分总和
total, err = db.Table("users").
    Where("status", 1).
    Sum("points")
```

### Avg - 平均值

```go
// 计算平均值
avg, err := db.Table("users").
    Where("status", 1).
    Avg("age")
fmt.Printf("平均年龄: %.2f\n", avg)

// 订单平均金额
avg, err = db.Table("orders").
    Where("status", "completed").
    Avg("amount")
```

### Max - 最大值
```go
// 最大值
max, err := db.Table("products").Max("price")
fmt.Printf("最高价格: %v\n", max)

// 最新时间
latest, err := db.Table("orders").Max("create_time")
fmt.Printf("最新订单时间: %v\n", latest)
```

### Min - 最小值
```go
// 最小值
min, err := db.Table("products").
    Where("status", 1).
    Min("price")
fmt.Printf("最低价格: %v\n", min)

// 最早注册时间
earliest, err := db.Table("users").Min("create_time")
```

## 分组聚合

### Group - 分组查询

```go
// 按状态分组统计
var results []map[string]interface{}
db.Table("users").
    Select("status", "COUNT(*) as count").
    Group("status").
    Get(&results)

for _, row := range results {
    fmt.Printf("状态: %v, 数量: %v\n", row["status"], row["count"])
}

// 按城市统计
db.Table("users").
    Select("city", "COUNT(*) as user_count", "AVG(age) as avg_age").
    Where("status", 1).
    Group("city").
    Get(&results)

for _, row := range results {
    fmt.Printf("城市: %v, 用户数: %v, 平均年龄: %v\n",
        row["city"], row["user_count"], row["avg_age"])
}
```

### Having - 分组过滤

```go
// 统计订单数大于 10 的用户
db.Table("orders").
    Select("user_id", "COUNT(*) as order_count").
    Group("user_id").
    Having("COUNT(*) > ?", 10).
    Get(&results)
```

## 组合示例

### 统计报表
```go
func GetUserStatistics() (map[string]interface{}, error) {
    stats := make(map[string]interface{})
    
    // 总用户数
    total, _ := db.Table("users").Count()
    stats["total"] = total
    
    // 激活用户
    active, _ := db.Table("users").Where("status", 1).Count()
    stats["active"] = active
    
    // 平均年龄
    avgAge, _ := db.Table("users").Where("status", 1).Avg("age")
    stats["avg_age"] = avgAge
    
    // 各城市用户数
    cities, _ := db.Table("users").
        Field("city, COUNT(*) as count").
        Where("status", 1).
        Group("city").
        Order("count DESC").
        Limit(10).
        Select()
    stats["top_cities"] = cities
    
    return stats, nil
}
```

### 销售统计
```go
func GetSalesStatistics(startDate, endDate string) (map[string]interface{}, error) {
    stats := make(map[string]interface{})
    
    // 订单总数
    count, _ := db.Table("orders").
        Where("status", "completed").
        WhereBetween("create_time", startDate, endDate).
        Count()
    stats["order_count"] = count
    
    // 销售总额
    total, _ := db.Table("orders").
        Where("status", "completed").
        WhereBetween("create_time", startDate, endDate).
        Sum("amount")
    stats["total_amount"] = total
    
    // 平均订单金额
    avg, _ := db.Table("orders").
        Where("status", "completed").
        WhereBetween("create_time", startDate, endDate).
        Avg("amount")
    stats["avg_amount"] = avg
    
    // 每日销售统计
    daily, _ := db.Table("orders").
        Field("DATE(create_time) as date, COUNT(*) as count, SUM(amount) as total").
        Where("status", "completed").
        WhereBetween("create_time", startDate, endDate).
        Group("DATE(create_time)").
        Order("date").
        Select()
    stats["daily"] = daily
    
    return stats, nil
}
```

## 注意事项
1. 聚合查询自动路由到从库
2. `Count` 返回 `int64` 类型
3. `Sum` 和 `Avg` 返回 `float64` 类型
4. `Max` 和 `Min` 返回 `interface{}` 类型，需要类型断言
5. 分组查询时注意使用正确的聚合函数
6. `Having` 用于分组后的条件过滤
