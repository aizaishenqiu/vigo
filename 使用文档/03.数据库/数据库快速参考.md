# 数据库快速参考手册

## 快速开始

### 创建 ORM 实例

```go
import "vigo/framework/db"

orm := db.NewORM(db.GlobalDB)
```

### 基础查询

```go
// 查询所有
var users []User
orm.Table("users").Find(&users)

// 查询单条
var user User
orm.Table("users").Where("id", 1).Scan(&user)

// 查询多条到 map
users, _ := orm.Table("users").Get()
```

## 常用方法速查

### 条件查询

```go
Where("status", 1)                    // 等于
Where("age", ">", 18)                 // 大于
WhereIn("id", []int{1, 2, 3})        // IN
WhereBetween("age", 18, 30)          // 范围
WhereNull("deleted_at")              // IS NULL
WhereNotNull("email")                // IS NOT NULL
WhereLike("name", "张")              // LIKE
WhereRaw("age > ?", 18)              // 原生条件
```

### 字段和表

```go
Table("users")                        // 指定表名
Select("id", "name")                 // 指定字段
Distinct()                           // 去重
```

### 连表查询

```go
LeftJoin("profiles", "users.id = profiles.user_id")
RightJoin("orders", "users.id = orders.user_id")
Join("posts", "users.id = posts.author_id", "INNER")
```

### 分组和排序

```go
Group("status")                       // 分组
Having("COUNT(*) > ?", 10)           // 分组过滤
OrderBy("created_at", "DESC")        // 排序
OrderByDesc("updated_at")            // 倒序
```

### 分页

```go
Limit(10)                             // 限制数量
Offset(20)                            // 偏移量
Page(1, 20)                          // 分页
```

### 执行方法

```go
Get()                                 // 获取所有
First()                               // 获取第一条
Find(&users)                         // 查询到结构体
Scan(&user)                          // 查询单条到结构体
Value("name")                        // 获取单个值
Pluck("email")                       // 获取单列
Count()                              // 统计
Sum("amount")                        // 求和
Avg("age")                           // 平均
Max("id")                            // 最大值
Min("age")                           // 最小值
```

### 数据操作

```go
Create(&user)                        // 创建记录
Insert(data)                         // 插入数据
InsertGetId(data)                    // 插入返回 ID
InsertMulti(count, data)            // 批量插入
Update(data)                         // 更新数据
Incr("views")                        // 字段自增
Decr("stock")                        // 字段自减
Delete()                             // 删除数据
```

### 高级功能

```go
Exists()                             // 检查是否存在
DoesntExist()                        // 检查是否不存在
Paginate(1, 20)                      // 分页查询
Chunk(100, callback)                 // 分块处理
ToSQL()                              // 获取 SQL
Dump()                               // 打印 SQL
Reset()                              // 重置查询
```

### 事务

```go
Transaction(func(tx *db.ORM) error {
    // 执行操作
    return nil  // nil 提交，error 回滚
})

tx, _ := orm.Begin()                 // 开启事务
tx.Commit()                          // 提交事务
tx.Rollback()                        // 回滚事务
```

### 调试

```go
Debug()                              // 调试模式
Context(ctx)                         // 设置上下文
```

## 常用示例

### 复杂查询

```go
var users []User
orm.Table("users u").
    LeftJoin("profiles p", "u.id = p.user_id").
    Select("u.id", "u.name", "p.bio").
    Where("u.status", 1).
    WhereIn("u.level", []int{1, 2, 3}).
    WhereBetween("u.age", 18, 30).
    WhereLike("u.name", "张").
    Group("u.status").
    Having("COUNT(*) > ?", 10).
    OrderByDesc("u.created_at").
    Limit(10).
    Find(&users)
```

### 分页查询

```go
result, _ := orm.Table("users").
    Where("status", 1).
    OrderByDesc("created_at").
    Paginate(1, 20)

fmt.Println("总数:", result.Total)
fmt.Println("当前页:", result.CurrentPage)
fmt.Println("数据:", result.Data)
```

### 事务处理

```go
err := orm.Transaction(func(tx *db.ORM) error {
    // 扣减余额
    tx.Table("users").
        Where("id", 1).
        Decr("balance", 100)
    
    // 创建订单
    tx.Table("orders").
        Insert(map[string]interface{}{
            "user_id": 1,
            "amount": 100,
        })
    
    return nil
})
```

### 批量操作

```go
// 批量插入
users := []map[string]interface{}{
    {"name": "user1", "email": "user1@example.com"},
    {"name": "user2", "email": "user2@example.com"},
}
orm.Table("users").InsertMulti(len(users), users)

// 批量更新
orm.Transaction(func(tx *db.ORM) error {
    for _, user := range users {
        tx.Where("id", user["id"]).
            Update(map[string]interface{}{
                "status": 1,
            })
    }
    return nil
})
```

### 聚合查询

```go
// 统计
count, _ := orm.Table("users").Count()

// 求和
total, _ := orm.Table("orders").Sum("amount")

// 平均
avg, _ := orm.Table("users").Avg("age")

// 分组统计
var results []map[string]interface{}
orm.Table("users").
    Select("status", "COUNT(*) as count").
    Group("status").
    Get(&results)
```

### 调试 SQL

```go
// 打印 SQL
orm.Debug().Table("users").Find(&users)

// 获取 SQL
sql := orm.Table("users").Where("id", 1).ToSQL()
fmt.Println("SQL:", sql)

// 打印最后执行的 SQL
orm.Table("users").Where("id", 1).Dump()
```

## 返回值类型

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `Get()` | `[]map[string]interface{}, error` | 所有记录 |
| `First()` | `map[string]interface{}, error` | 第一条记录 |
| `Find(&dest)` | `error` | 填充到结构体切片 |
| `Scan(&dest)` | `error` | 填充到单个结构体 |
| `Value(col)` | `interface{}, error` | 单个值 |
| `Pluck(col)` | `[]interface{}, error` | 单列值 |
| `Count()` | `int64, error` | 数量 |
| `Sum(col)` | `float64, error` | 求和 |
| `Avg(col)` | `float64, error` | 平均 |
| `Insert()` | `int64, error` | 影响行数 |
| `InsertGetId()` | `int64, error` | 插入的 ID |
| `Update()` | `int64, error` | 影响行数 |
| `Delete()` | `int64, error` | 删除行数 |

```go
    Email string `db:"email"`

```

```go
err := orm.Transaction(func(tx *db.ORM) error {

    return nil
```

```go
result, _ := orm.Paginate(1, 20)
```

```go
orm.Chunk(100, func(rows []map[string]interface{}) error {
```

```go
orm.Debug().Table("users").Find(&users)
```

### 6. 使用 Select 指定字段

```go
orm.Select("id", "name", "email").Find(&users)
```

## 常见错误

### 1. 忘记传递结构体指针

```go
// ❌ 错误
var user User
orm.Table("users").Where("id", 1).Scan(user)

// ✅ 正确
var user User
orm.Table("users").Where("id", 1).Scan(&user)
```

### 2. 忘记处理错误

```go
// ❌ 错误
orm.Table("users").Find(&users)

// ✅ 正确
err := orm.Table("users").Find(&users)
if err != nil {
    log.Fatal(err)
}
```

### 3. 更新/删除不指定条件

```go
// ❌ 错误 - 会更新/删除所有记录
orm.Table("users").Update(data)

// ✅ 正确
orm.Table("users").Where("id", 1).Update(data)
```

## 性能优化

### 1. 避免深度分页

```go
// ❌ 性能差
orm.Page(10000, 20)

// ✅ 使用游标
orm.Where("id", ">", lastID).Limit(20)
```

### 2. 使用覆盖索引

```go
// ✅ 只查询需要的字段
orm.Select("id", "name", "email").Find(&users)
```

### 3. 使用原子操作

```go
// ✅ 避免并发问题
orm.Where("id", 1).Incr("views")
```

### 4. 使用分块处理大数据

```go
// ✅ 避免内存溢出
orm.Chunk(1000, callback)
```
