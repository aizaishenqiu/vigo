# 查询构造器

## 简介
查询构造器是框架的核心组件，提供 ThinkPHP/Laravel 风格的链式操作，让数据库查询更加简洁直观。

## 创建查询构造器

### 基本方式
```go
import "vigo/framework/db"

// 方式1: 使用 NewQuery
query := db.NewQuery(db.GlobalDB).Table("users")

// 方式2: 使用门面方法 Table
query := db.Table("users")

// 方式3: 使用别名方法 Name
query := db.Name("users")
```

## 链式操作方法

### Table - 指定表名
```go
// 指定表名
query := db.Table("users")

// 使用别名
query := db.Table("users u")
query := db.Table("users").Alias("u")
```

### Field - 指定查询字段
```go
// 字符串方式
query := db.Table("users").Field("id, name, email")

// 数组方式
query := db.Table("users").Field([]string{"id", "name", "email"})

// 默认查询所有字段
query := db.Table("users")  // SELECT *
```

### Where - 条件查询
```go
// 等于条件
query := db.Table("users").Where("status", 1)

// 比较条件
query := db.Table("users").Where("age", ">", 18)
query := db.Table("users").Where("age", ">=", 18)
query := db.Table("users").Where("age", "<", 65)

// 多条件
query := db.Table("users").
    Where("status", 1).
    Where("age", ">", 18).
    Where("city", "北京")
```

### WhereIn - IN 条件
```go
// IN 条件
query := db.Table("users").WhereIn("id", []interface{}{1, 2, 3})

// NOT IN 条件
query := db.Table("users").WhereNotIn("status", []interface{}{0, -1})
```

### WhereBetween - BETWEEN 条件
```go
// BETWEEN 条件
query := db.Table("users").WhereBetween("age", 18, 65)

// NOT BETWEEN 条件
query := db.Table("users").WhereNotBetween("age", 0, 17)
```

### WhereNull - 空值条件
```go
// IS NULL
query := db.Table("users").WhereNull("delete_time")

// IS NOT NULL
query := db.Table("users").WhereNotNull("email")
```

### WhereLike - 模糊查询
```go
// LIKE 条件（自动添加 %%）
query := db.Table("users").WhereLike("name", "张")
// 生成: WHERE name LIKE '%张%'
```

### WhereRaw - 原生条件
```go
// 原生 SQL 条件
query := db.Table("users").
    WhereRaw("age > ? AND status = ?", 18, 1)

// OR 条件
query := db.Table("users").
    WhereRaw("status = 1").
    WhereOrRaw("vip = 1")
```

### WhereMap - 批量条件
```go
// 批量设置条件
conditions := map[string]interface{}{
    "status": 1,
    "city":   "北京",
}
query := db.Table("users").WhereMap(conditions)
```

### Order - 排序
```go
// 单字段排序
query := db.Table("users").Order("create_time DESC")

// 多字段排序
query := db.Table("users").
    Order("status ASC").
    Order("create_time DESC")
```

### Group - 分组
```go
// 分组查询
query := db.Table("users").
    Field("city, COUNT(*) as count").
    Group("city")
```

### Having - 分组条件
```go
// 分组后过滤
query := db.Table("users").
    Field("city, COUNT(*) as count").
    Group("city").
    Having("COUNT(*) > 100")
```

### Join - 连表查询
```go
// INNER JOIN
query := db.Table("orders o").
    Join("users u", "o.user_id = u.id", "INNER").
    Field("o.*, u.name")

// LEFT JOIN
query := db.Table("orders o").
    Join("users u", "o.user_id = u.id", "LEFT").
    Field("o.*, u.name")

// 多表连接
query := db.Table("orders o").
    Join("users u", "o.user_id = u.id", "INNER").
    Join("products p", "o.product_id = p.id", "INNER").
    Field("o.*, u.name, p.title")
```

### Limit - 限制数量
```go
// 限制返回数量
query := db.Table("users").Limit(10)
```

### Page - 分页
```go
// 分页（第1页，每页10条）
query := db.Table("users").Page(1, 10)
```

### Distinct - 去重
```go
// 去重查询
query := db.Table("users").Distinct()
```

### Lock - 锁定
```go
// 排他锁 FOR UPDATE
query := db.Table("users").Lock()

// 共享锁 LOCK IN SHARE MODE
query := db.Table("users").Lock("LOCK IN SHARE MODE")
```

## 执行方法

### Find - 查询单条
```go
user, err := db.Table("users").Where("id", 1).Find()
if err != nil {
    log.Fatal(err)
}
if user != nil {
    fmt.Println(user["name"])
}
```

### Select - 查询多条
```go
users, err := db.Table("users").
    Where("status", 1).
    Order("id DESC").
    Select()
if err != nil {
    log.Fatal(err)
}
for _, user := range users {
    fmt.Println(user["name"])
}
```

### Value - 获取单个值
```go
name, err := db.Table("users").Where("id", 1).Value("name")
if err != nil {
    log.Fatal(err)
}
fmt.Println(name)
```

### Column - 获取列值
```go
emails, err := db.Table("users").Column("email")
if err != nil {
    log.Fatal(err)
}
for _, email := range emails {
    fmt.Println(email)
}
```

### Count - 统计数量
```go
count, err := db.Table("users").Where("status", 1).Count()
fmt.Printf("用户数量: %d\n", count)
```

### Insert - 插入数据
```go
id, err := db.Table("users").InsertGetId(map[string]interface{}{
    "name":  "张三",
    "email": "zhangsan@example.com",
})
```

### Update - 更新数据
```go
affected, err := db.Table("users").
    Where("id", 1).
    Update(map[string]interface{}{
        "name": "李四",
    })
```

### Delete - 删除数据
```go
affected, err := db.Table("users").Where("id", 1).Delete()
```

## 获取SQL

### GetLastSql - 获取最后执行的SQL
```go
query := db.Table("users").Where("id", 1)
user, _ := query.Find()
sql := query.GetLastSql()
fmt.Println(sql)  // SELECT * FROM users WHERE id = 1 LIMIT 1
```

## 注意事项
1. 链式方法返回 `*Query`，可继续链式调用
2. 执行方法（Find、Select等）才会真正执行查询
3. 条件方法会累积，形成 AND 条件
4. 所有查询都使用预处理语句，防止SQL注入
5. 查询操作自动路由到从库，写入操作路由到主库
