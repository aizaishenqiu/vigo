# ORM 使用指南

## 简介

Vigo ORM 参考了 Beego ORM、GORM、Laravel Eloquent 的设计理念，提供了一套完整、易用、高性能的数据库操作方案。

### 核心特性

- ✅ **链式调用** - 流畅的链式 API，代码更简洁
- ✅ **类型转换** - 智能类型转换，支持各种数据类型
- ✅ **事务支持** - 闭包事务和手动事务
- ✅ **关联查询** - 支持连表查询
- ✅ **聚合函数** - Count/Sum/Avg/Max/Min
- ✅ **分页查询** - 结构化分页结果
- ✅ **调试模式** - SQL 打印和性能分析
- ✅ **批量操作** - 批量插入、批量更新

## 快速开始

### 1. 基础查询

```go
import "vigo/framework/db"

// 创建 ORM 实例
orm := db.NewORM(db.GlobalDB)

// 查询所有记录
var users []User
orm.Table("users").Find(&users)

// 查询单条记录
var user User
orm.Table("users").Where("id", 1).Scan(&user)

// 查询指定字段
var users []User
orm.Table("users").
    Select("id", "username", "email").
    Find(&users)
```

### 2. 条件查询

```go
// 基础条件
orm.Where("status", 1)

// 多条件组合
orm.Where("status", 1).
    Where("age", ">=", 18).
    Where("city", "北京")

// IN 条件
orm.WhereIn("id", []int{1, 2, 3})
orm.WhereNotIn("status", []int{0})

// 范围条件
orm.WhereBetween("age", 18, 30)
orm.WhereNotBetween("score", 0, 60)

// NULL 条件
orm.WhereNull("deleted_at")
orm.WhereNotNull("email")

// 模糊查询
orm.WhereLike("username", "john")

// OR 条件
orm.Where("status", 1).
    WhereOr("status", 2)

// 原生条件
orm.WhereRaw("age > ?", 18)
orm.WhereOrRaw("status = ?", 1)

// 批量条件
orm.WhereMap(map[string]interface{}{
    "status": 1,
    "age":    18,
})
```

### 3. 连表查询

```go
// 左连接
var users []User
orm.Table("users").
    LeftJoin("profiles", "users.id = profiles.user_id").
    Where("profiles.age", ">", 18).
    Find(&users)

// 右连接
orm.Table("users").
    RightJoin("orders", "users.id = orders.user_id").
    Find(&users)

// 内连接
orm.Table("users").
    Join("posts", "users.id = posts.author_id", "INNER").
    Find(&users)
```

### 4. 分组和聚合

```go
// 分组
var results []map[string]interface{}
orm.Table("orders").
    Select("user_id", "SUM(amount) as total").
    Group("user_id").
    Get(&results)

// 分组过滤
orm.Table("orders").
    Select("user_id", "COUNT(*) as count").
    Group("user_id").
    Having("COUNT(*) > ?", 10).
    Get(&results)

// 聚合函数
count, _ := orm.Table("users").Count()
sum, _ := orm.Table("orders").Sum("amount")
avg, _ := orm.Table("scores").Avg("score")
max, _ := orm.Table("users").Max("id")
min, _ := orm.Table("users").Min("age")
```

### 5. 排序和分页

```go
// 排序
orm.OrderBy("created_at", "DESC")
orm.OrderByDesc("updated_at")
orm.Order("id ASC")

// 简单分页
orm.Limit(10).Offset(20)

// 分页方法
orm.Page(1, 20)

// 结构化分页
result, _ := orm.Table("users").
    Where("status", 1).
    Paginate(1, 20)

fmt.Println("Total:", result.Total)
fmt.Println("Current Page:", result.CurrentPage)
fmt.Println("Last Page:", result.LastPage)
fmt.Println("Data:", result.Data)
```

### 6. 数据操作

#### 插入数据

```go
// 插入单条
data := map[string]interface{}{
    "username": "john",
    "email":    "john@example.com",
    "status":   1,
}
id, _ := orm.Table("users").InsertGetId(data)
fmt.Println("Inserted ID:", id)

// 批量插入
users := []map[string]interface{}{
    {"username": "user1", "email": "user1@example.com"},
    {"username": "user2", "email": "user2@example.com"},
    {"username": "user3", "email": "user3@example.com"},
}
rows, _ := orm.Table("users").InsertMulti(len(users), users)
fmt.Println("Inserted rows:", rows)

// 使用模型创建
type User struct {
    Id       int64  `db:"id"`
    Username string `db:"username"`
    Email    string `db:"email"`
}

user := &User{
    Username: "jane",
    Email:    "jane@example.com",
}
err := orm.Table("users").Create(user)
```

#### 更新数据

```go
// 更新单条
data := map[string]interface{}{
    "status": 1,
    "phone":  "13800138000",
}
rows, _ := orm.Table("users").
    Where("id", 1).
    Update(data)
fmt.Println("Updated rows:", rows)

// 字段自增
rows, _ := orm.Table("posts").
    Where("id", 1).
    Incr("view_count")

// 字段自增（指定步长）
rows, _ := orm.Table("posts").
    Where("id", 1).
    Incr("view_count", 5)

// 字段自减
rows, _ := orm.Table("posts").
    Where("id", 1).
    Decr("view_count")

// 字段自减（指定步长）
rows, _ := orm.Table("posts").
    Where("id", 1).
    Decr("view_count", 3)
```

#### 删除数据

```go
// 删除单条
rows, _ := orm.Table("users").
    Where("id", 1).
    Delete()

// 批量删除
rows, _ := orm.Table("users").
    WhereIn("id", []int{1, 2, 3}).
    Delete()
```

### 7. 事务处理

#### 闭包事务（推荐）

```go
err := orm.Transaction(func(tx *db.ORM) error {
    // 创建用户
    userId, err := tx.Table("users").InsertGetId(userData)
    if err != nil {
        return err
    }
    
    // 创建用户资料
    profileData["user_id"] = userId
    _, err = tx.Table("user_profiles").Insert(profileData)
    if err != nil {
        return err
    }
    
    return nil
})

if err != nil {
    fmt.Println("Transaction failed:", err)
} else {
    fmt.Println("Transaction succeeded")
}
```

#### 手动事务

```go
tx, err := orm.Begin()
if err != nil {
    fmt.Println("Begin failed:", err)
    return
}

defer func() {
    if p := recover(); p != nil {
        tx.Rollback()
        panic(p)
    }
}()

// 执行多个操作
tx.Table("users").Insert(userData)
tx.Table("profiles").Insert(profileData)

// 提交事务
err = tx.Commit()
if err != nil {
    fmt.Println("Commit failed:", err)
    tx.Rollback()
}
```

### 8. 高级功能

#### 检查是否存在

```go
exists, _ := orm.Table("users").
    Where("email", "test@example.com").
    Exists()

if exists {
    fmt.Println("User exists")
} else {
    fmt.Println("User not exists")
}

// 检查是否不存在
notExists, _ := orm.Table("users").
    Where("email", "test@example.com").
    DoesntExist()
```

#### 获取单个值

```go
// 获取单个字段值
email, _ := orm.Table("users").
    Where("id", 1).
    Value("email")

// 获取单列值
emails, _ := orm.Table("users").
    Where("status", 1).
    Pluck("email")
```

#### 分块处理

```go
// 处理大数据集
err := orm.Table("users").
    Where("status", 1).
    Chunk(100, func(rows []map[string]interface{}) error {
        for _, row := range rows {
            // 处理每批 100 条数据
            fmt.Println("Processing:", row["username"])
        }
        return nil
    })

if err != nil {
    fmt.Println("Chunk processing failed:", err)
}
```

#### 去重查询

```go
// DISTINCT 查询
var emails []interface{}
orm.Table("users").
    Distinct().
    Pluck("email")
```

#### 加锁查询

```go
// 排他锁（SELECT ... FOR UPDATE）
var user User
orm.Table("users").
    Where("id", 1).
    LockForUpdate().
    Scan(&user)

// 共享锁（SELECT ... LOCK IN SHARE MODE）
orm.Table("users").
    Where("id", 1).
    Lock("LOCK IN SHARE MODE").
    Scan(&user)
```

### 9. 调试功能

#### 开启调试模式

```go
// 调试模式会打印 SQL 和参数
orm.Debug().
    Table("users").
    Where("status", 1).
    Find(&users)
```

#### 打印 SQL

```go
// 打印最后执行的 SQL
orm.Table("users").
    Where("id", 1).
    Dump()

// 输出:
// SQL: SELECT * FROM users WHERE id = ?
// Args: [1]
```

#### 获取 SQL（不执行）

```go
sql := orm.Table("users").
    Where("id", 1).
    ToSQL()

fmt.Println("SQL:", sql)
```

#### 设置上下文和超时

```go
ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
defer cancel()

orm.Context(ctx).
    Timeout(30 * time.Second).
    Table("users").
    Find(&users)
```

### 10. 重置查询条件

```go
// 重置后可以重新构建查询
orm.Table("users").
    Where("status", 1).
    Reset().  // 重置所有条件
    Table("posts").
    Where("id", 1).
    Scan(&post)
```

## 完整示例

### 用户管理示例

```go
package main

import (
    "fmt"
    "vigo/framework/db"
)

type User struct {
    Id       int64  `json:"id" db:"id"`
    Username string `json:"username" db:"username"`
    Email    string `json:"email" db:"email"`
    Status   int    `json:"status" db:"status"`
}

func main() {
    orm := db.NewORM(db.GlobalDB)
    
    // 创建用户
    userData := map[string]interface{}{
        "username": "john_doe",
        "email":    "john@example.com",
        "status":   1,
    }
    userId, _ := orm.Table("users").InsertGetId(userData)
    fmt.Println("Created user ID:", userId)
    
    // 查询用户
    var user User
    err := orm.Table("users").
        Where("id", userId).
        Scan(&user)
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    fmt.Println("User:", user.Username, user.Email)
    
    // 更新用户
    _, err = orm.Table("users").
        Where("id", userId).
        Update(map[string]interface{}{
            "status": 1,
        })
    
    // 查询列表
    var users []User
    err = orm.Table("users").
        Where("status", 1).
        OrderByDesc("created_at").
        Limit(10).
        Find(&users)
    
    // 分页查询
    result, _ := orm.Table("users").
        Where("status", 1).
        Paginate(1, 20)
    fmt.Printf("Total: %d, Page: %d\n", result.Total, result.CurrentPage)
    
    // 统计
    count, _ := orm.Table("users").
        Where("status", 1).
        Count()
    fmt.Println("Total users:", count)
}
```

### 复杂查询示例

```go
func GetActiveUsers() ([]User, error) {
    orm := db.NewORM(db.GlobalDB)
    
    var users []User
    err := orm.Table("users").
        Select("id", "username", "email", "status").
        Where("status", 1).
        WhereBetween("age", 18, 30).
        WhereLike("username", "john").
        WhereOr(func(q *db.Query) {
            q.Where("email", "like", "%@example.com")
        }).
        LeftJoin("profiles", "users.id = profiles.user_id").
        Group("users.status").
        Having("COUNT(*) > ?", 1).
        OrderByDesc("created_at").
        Limit(10).
        Find(&users)
    
    return users, err
}
```

### 事务示例

```go
func CreateUserWithProfile(userData, profileData map[string]interface{}) error {
    orm := db.NewORM(db.GlobalDB)
    
    return orm.Transaction(func(tx *db.ORM) error {
        // 创建用户
        userId, err := tx.Table("users").InsertGetId(userData)
        if err != nil {
            return err
        }
        
        // 创建资料
        profileData["user_id"] = userId
        _, err = tx.Table("user_profiles").Insert(profileData)
        if err != nil {
            return err
        }
        
        return nil
    })
}
```

## 最佳实践

### 1. 使用模型定义

```go
type User struct {
    Id        int64     `json:"id" db:"id"`
    Username  string    `json:"username" db:"username"`
    Email     string    `json:"email" db:"email"`
    CreatedAt time.Time `json:"created_at" db:"created_at"`
}

// 使用
var user User
orm.Table("users").Where("id", 1).Scan(&user)
```

### 2. 使用事务保证数据一致性

```go
err := orm.Transaction(func(tx *db.ORM) error {
    // 多个操作要么都成功，要么都失败
    tx.Table("users").Insert(userData)
    tx.Table("profiles").Insert(profileData)
    return nil
})
```

### 3. 使用分页避免大数据量

```go
result, err := orm.Paginate(1, 20)
```

### 4. 使用 Chunk 处理超大数据集

```go
orm.Chunk(100, func(rows []map[string]interface{}) error {
    // 每批处理 100 条
    return nil
})
```

### 5. 使用调试模式优化 SQL

```go
orm.Debug().Table("users").Find(&users)
```

### 6. 使用 Select 指定字段

```go
// 好的做法
orm.Select("id", "username", "email").Find(&users)

// 避免
orm.Find(&users)  // SELECT *
```

## 常见问题

### Q: 如何处理 NULL 值？

```go
// 使用指针类型
type User struct {
    Phone *string `db:"phone"`
}

// 或者使用 sql.NullString
type User struct {
    Phone sql.NullString `db:"phone"`
}
```

### Q: 如何自定义表名？

```go
orm.Table("my_users").Find(&users)
```

### Q: 如何实现软删除？

```go
type User struct {
    DeletedAt *time.Time `db:"deleted_at"`
}

// 查询时自动过滤已删除
orm.WhereNull("deleted_at").Find(&users)

// 软删除
orm.Where("id", 1).
    Update(map[string]interface{}{
        "deleted_at": time.Now(),
    })
```

### Q: 如何批量更新？

```go
// 方法 1: 使用事务
orm.Transaction(func(tx *db.ORM) error {
    for _, user := range users {
        tx.Where("id", user.Id).Update(user.Data)
    }
    return nil
})

// 方法 2: 使用 CASE WHEN
// 需要手动编写 SQL
```
