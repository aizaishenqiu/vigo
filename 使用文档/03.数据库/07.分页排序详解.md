# 分页排序详解

## 简介

分页和排序是数据库查询中最常用的功能，框架提供简洁的分页和排序方法。

## 排序操作

### OrderBy - 单字段排序

```go
// 降序排列
var users []User
db.Table("users").
    OrderByDesc("create_time").
    Find(&users)

// 升序排列
db.Table("users").
    OrderBy("id", "ASC").
    Find(&users)
```

### Order - 多字段排序

```go
// 多字段排序
db.Table("users").
    OrderBy("status", "ASC").
    OrderBy("create_time", "DESC").
    Find(&users)
// 生成：ORDER BY status ASC, create_time DESC

// 复杂排序
var orders []Order
db.Table("orders").
    OrderByDesc("priority").
    OrderBy("create_time", "ASC").
    Find(&orders)
```

### OrderByDesc - 倒序排列

```go
// 按创建时间倒序
db.Table("users").
    OrderByDesc("created_at").
    Find(&users)

// 按更新时间倒序
db.Table("articles").
    OrderByDesc("updated_at").
    Find(&articles)
```

## 分页操作

### Limit - 限制数量

```go
// 限制返回 10 条
var users []User
db.Table("users").
    OrderByDesc("id").
    Limit(10).
    Find(&users)
```

### Offset - 偏移量

```go
// 跳过前 10 条，取后 10 条
var users []User
db.Table("users").
    OrderByDesc("id").
    Limit(10).
    Offset(10).
    Find(&users)
// 生成：LIMIT 10 OFFSET 10
```

### Page - 分页方法

```go
// 第 1 页，每页 10 条
var users []User
db.Table("users").
    OrderByDesc("id").
    Page(1, 10).
    Find(&users)

// 第 2 页，每页 10 条
db.Table("users").
    OrderByDesc("id").
    Page(2, 10).
    Find(&users)
// 生成：LIMIT 10 OFFSET 10

// 第 3 页，每页 20 条
db.Table("users").
    OrderByDesc("id").
    Page(3, 20).
    Find(&users)
// 生成：LIMIT 20 OFFSET 40
```

## 分页计算

### 分页公式

```go
// Page(page, pageSize) 内部计算
// LIMIT pageSize OFFSET (page - 1) * pageSize

// 示例：
// Page(1, 10) => LIMIT 10 OFFSET 0
// Page(2, 10) => LIMIT 10 OFFSET 10
// Page(3, 10) => LIMIT 10 OFFSET 20
```

## 结构化分页

### Paginate - 完整分页结果

```go
// 获取分页结果
result, err := db.Table("users").
    Where("status", 1).
    OrderByDesc("created_at").
    Paginate(1, 20)

if err != nil {
    log.Fatal(err)
}

// 分页信息
fmt.Printf("总数：%d\n", result.Total)
fmt.Printf("当前页：%d\n", result.CurrentPage)
fmt.Printf("每页数量：%d\n", result.PerPage)
fmt.Printf("总页数：%d\n", result.LastPage)
fmt.Printf("是否有上一页：%v\n", result.HasMorePages())
fmt.Printf("是否有下一页：%v\n", result.HasMorePages())

// 数据
users := result.Data
```

### 分页结果结构

```go
type PaginateResult struct {
    Total       int64                    `json:"total"`        // 总数
    CurrentPage int                      `json:"current_page"` // 当前页
    PerPage     int                      `json:"per_page"`     // 每页数量
    LastPage    int                      `json:"last_page"`    // 总页数
    Data        []map[string]interface{} `json:"data"`         // 数据
}
```

## 分块处理

### Chunk - 大数据集处理

```go
// 处理大数据集
err := db.Table("users").
    Where("status", 1).
    Chunk(100, func(rows []map[string]interface{}) error {
        for _, row := range rows {
            // 处理每批 100 条数据
            fmt.Printf("处理用户：%v\n", row["name"])
        }
        return nil
    })

if err != nil {
    log.Fatal(err)
}
```

## 完整示例

### 用户列表分页

```go
func GetUserList(page, pageSize int) (*db.PaginateResult, error) {
    return db.Table("users").
        Where("status", 1).
        OrderByDesc("created_at").
        Paginate(page, pageSize)
}
```

### 商品列表（带排序和分页）

```go
func GetProductList(page, pageSize, sortType int) (*db.PaginateResult, error) {
    query := db.Table("products").Where("status", 1)
    
    // 根据排序类型排序
    switch sortType {
    case 1: // 价格从低到高
        query.OrderBy("price", "ASC")
    case 2: // 价格从高到低
        query.OrderByDesc("price")
    case 3: // 销量从高到低
        query.OrderByDesc("sales")
    default: // 最新上架
        query.OrderByDesc("created_at")
    }
    
    return query.Paginate(page, pageSize)
}
```

### 导出大量数据

```go
func ExportUsers(filename string) error {
    file, err := os.Create(filename)
    if err != nil {
        return err
    }
    defer file.Close()
    
    // 写入 CSV 表头
    file.WriteString("ID,Name,Email\n")
    
    // 分块处理
    return db.Table("users").
        Where("status", 1).
        Chunk(1000, func(rows []map[string]interface{}) error {
            for _, row := range rows {
                line := fmt.Sprintf("%v,%s,%s\n",
                    row["id"], row["name"], row["email"])
                file.WriteString(line)
            }
            return nil
        })
}
```

## 注意事项

1. 分页页码从 1 开始
2. 使用 `Paginate` 可以获取完整的分页信息
3. 大数据集使用 `Chunk` 避免内存溢出
4. 分页查询建议配合索引使用
5. 深度分页（大 OFFSET）会影响性能

## 性能优化

### 避免深度分页

```go
// 不推荐：深度分页，性能差
db.Table("users").Page(10000, 20)

// 推荐：使用游标分页
db.Table("users").
    Where("id", ">", lastID).
    OrderBy("id", "ASC").
    Limit(20)
```

### 使用覆盖索引

```go
// 只查询需要的字段
db.Table("users").
    Select("id", "name", "email").
    Where("status", 1).
    Page(1, 20)
```
