# 写入操作详解

## 简介

写入操作包括插入、更新、删除等操作，框架提供完整的写入方法支持。

## 插入操作

### Insert - 插入记录

```go
// 插入记录，返回影响行数
affected, err := db.Table("users").Insert(map[string]interface{}{
    "name":        "张三",
    "email":       "zhangsan@example.com",
    "status":      1,
    "create_time": time.Now().Format("2006-01-02 15:04:05"),
})

if err != nil {
    log.Fatal(err)
}
fmt.Printf("插入成功，影响行数：%d\n", affected)
```

### InsertGetId - 插入并返回ID

```go
// 插入记录并获取自增 ID
id, err := db.Table("users").InsertGetId(map[string]interface{}{
    "name":        "李四",
    "email":       "lisi@example.com",
    "status":      1,
    "create_time": time.Now().Format("2006-01-02 15:04:05"),
})

if err != nil {
    log.Fatal(err)
}
fmt.Printf("插入成功，ID: %d\n", id)
```

### InsertMulti - 批量插入

```go
// 批量插入
users := []map[string]interface{}{
    {
        "name":        "用户 1",
        "email":       "user1@example.com",
        "status":      1,
        "create_time": time.Now().Format("2006-01-02 15:04:05"),
    },
    {
        "name":        "用户 2",
        "email":       "user2@example.com",
        "status":      1,
        "create_time": time.Now().Format("2006-01-02 15:04:05"),
    },
    {
        "name":        "用户 3",
        "email":       "user3@example.com",
        "status":      1,
        "create_time": time.Now().Format("2006-01-02 15:04:05"),
    },
}

affected, err := db.Table("users").InsertMulti(len(users), users)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("批量插入成功，影响行数：%d\n", affected)
```

### Create - 创建记录（使用结构体）

```go
type User struct {
    Id        int64     `db:"id"`
    Name      string    `db:"name"`
    Email     string    `db:"email"`
    Status    int       `db:"status"`
    CreatedAt time.Time `db:"create_time"`
}

user := &User{
    Name:      "王五",
    Email:     "wangwu@example.com",
    Status:    1,
    CreatedAt: time.Now(),
}

err := db.Table("users").Create(user)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("创建成功，ID: %d\n", user.Id)
```

## 更新操作

### Update - 更新记录

```go
// 更新记录
affected, err := db.Table("users").
    Where("id", 1).
    Update(map[string]interface{}{
        "name":       "新名字",
        "email":      "newemail@example.com",
        "update_time": time.Now().Format("2006-01-02 15:04:05"),
    })

if err != nil {
    log.Fatal(err)
}
fmt.Printf("更新成功，影响行数: %d\n", affected)
```

### Incr - 字段自增

```go
// 自增 1（默认）
affected, err := db.Table("users").
    Where("id", 1).
    Incr("points")

// 自增指定步长
affected, err = db.Table("users").
    Where("id", 1).
    Incr("points", 10)
```

### Decr - 字段自减

```go
// 自减 1（默认）
affected, err := db.Table("products").
    Where("id", 1).
    Decr("stock")

// 自减指定步长
affected, err = db.Table("products").
    Where("id", 1).
    Decr("stock", 5)
```

## 删除操作

### Delete - 删除记录

```go
// 物理删除
affected, err := db.Table("users").
    Where("id", 1).
    Delete()

if err != nil {
    log.Fatal(err)
}
fmt.Printf("删除成功，影响行数: %d\n", affected)
```

### 软删除实现

```go
// 软删除（更新 deleted_at 字段）
affected, err := db.Table("users").
    Where("id", 1).
    Update(map[string]interface{}{
        "deleted_at": time.Now().Format("2006-01-02 15:04:05"),
    })

// 查询时排除已删除记录
var users []User
db.Table("users").
    WhereNull("deleted_at").
    Find(&users)
```

## 条件更新

### 带条件的更新

```go
// 更新指定状态的用户
affected, err := db.Table("users").
    Where("status", 0).
    Where("create_time", "<", "2024-01-01").
    Update(map[string]interface{}{
        "status": -1,
    })
```

### 带条件的删除

```go
// 删除过期数据
affected, err := db.Table("logs").
    Where("create_time", "<", "2023-01-01").
    Delete()
```

## 完整示例

### 创建订单

```go
func CreateOrder(userID, productID int64, quantity int, amount float64) (int64, error) {
    orderID, err := db.Table("orders").InsertGetId(map[string]interface{}{
        "user_id":    userID,
        "product_id": productID,
        "quantity":   quantity,
        "amount":     amount,
        "status":     "pending",
        "create_time": time.Now().Format("2006-01-02 15:04:05"),
    })

    if err != nil {
        return 0, err
    }

    // 扣减库存
    _, err = db.Table("products").
        Where("id", productID).
        Decr("stock", quantity)

    if err != nil {
        return 0, err
    }

    return orderID, nil
}
```

### 更新用户状态

```go
func UpdateUserStatus(userID int64, status int) error {
    affected, err := db.Table("users").
        Where("id", userID).
        Update(map[string]interface{}{
            "status":     status,
            "update_time": time.Now().Format("2006-01-02 15:04:05"),
        })

    if err != nil {
        return err
    }

    if affected == 0 {
        return fmt.Errorf("用户不存在")
    }

    return nil
}
```

## 注意事项

1. 写入操作自动路由到主库
2. 更新和删除操作必须指定条件，避免全表操作
3. 批量插入时注意数据量，避免单次插入过多
4. 重要操作前做好数据备份
5. 使用事务保证数据一致性
6. 使用 `Incr`/`Decr` 进行原子操作，避免并发问题

### 使用事务批量插入

```go
func BatchInsertUsers(users []map[string]interface{}) error {
    orm := db.NewORM(db.GlobalDB)
    
    return orm.Transaction(func(tx *db.ORM) error {
        _, err := tx.Table("users").InsertMulti(len(users), users)
        return err
    })
}
```
