# 查询操作详解

## 简介

框架提供丰富的查询方法，支持单条查询、多条查询、条件查询、聚合查询等。

## 单条查询

### Scan - 查询单条记录到结构体

```go
// 根据 ID 查询
var user User
err := db.Table("users").Where("id", 1).Scan(&user)

// 根据条件查询
var user User
err = db.Table("users").
    Where("email", "test@example.com").
    Scan(&user)

// 检查结果
if err != nil {
    log.Fatal(err)
}
fmt.Printf("用户名：%v, 邮箱：%v\n", user.Name, user.Email)
```

### First - 查询单条记录

```go
// 查询第一条记录
user, err := db.Table("users").
    Where("status", 1).
    First()

if err != nil {
    log.Fatal(err)
}
if user == nil {
    fmt.Println("记录不存在")
} else {
    fmt.Printf("用户名：%v\n", user["name"])
}
```

### Find - 查询到结构体

```go
// 查询单条到结构体
var user User
err := db.Table("users").Where("id", 1).Find(&user)

// 查询多条到结构体
var users []User
err = db.Table("users").Where("status", 1).Find(&users)
```

### Value - 获取单个字段值

```go
// 获取单个字段
name, err := db.Table("users").Where("id", 1).Value("name")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("用户名：%v\n", name)
```

## 多条查询

### Get - 查询多条记录

```go
// 查询所有记录
users, err := db.Table("users").Get()

// 条件查询
users, err = db.Table("users").
    Where("status", 1).
    OrderByDesc("create_time").
    Get()

// 遍历结果
for _, user := range users {
    fmt.Printf("ID: %v, 姓名：%v\n", user["id"], user["name"])
}
```

### Pluck - 获取单列值

```go
// 获取所有用户邮箱
emails, err := db.Table("users").
    Where("status", 1).
    Pluck("email")

if err != nil {
    log.Fatal(err)
}

for _, email := range emails {
    fmt.Printf("邮箱：%v\n", email)
}
```

## 条件查询

### Where - 基本条件

```go
// 等于
db.Table("users").Where("status", 1)

// 比较条件
db.Table("users").Where("age", ">", 18)
db.Table("users").Where("age", ">=", 18)
db.Table("users").Where("age", "<", 65)
db.Table("users").Where("age", "<=", 65)

// 不等于
db.Table("users").Where("status", "<>", 0)

// 多条件链式调用
db.Table("users").
    Where("status", 1).
    Where("age", ">=", 18).
    Where("city", "北京")
```

### WhereIn - IN 条件

```go
// IN 条件
db.Table("users").
    WhereIn("id", []int{1, 2, 3, 4, 5})

// NOT IN 条件
db.Table("users").
    WhereNotIn("status", []int{0, -1})
```

### WhereBetween - 范围条件

```go
// BETWEEN 条件
db.Table("users").
    WhereBetween("age", 18, 65)

// NOT BETWEEN 条件
db.Table("users").
    WhereNotBetween("age", 0, 17)

// 日期范围
db.Table("orders").
    WhereBetween("create_time", "2024-01-01", "2024-12-31")
```

### WhereNull - 空值条件

```go
// IS NULL
db.Table("users").
    WhereNull("delete_time")

// IS NOT NULL
db.Table("users").
    WhereNotNull("email")
```

### WhereLike - 模糊查询

```go
// LIKE 条件（自动添加 %%）
db.Table("users").
    WhereLike("name", "张")
// 生成：WHERE name LIKE '%张%'

// 邮箱域名查询
db.Table("users").
    WhereLike("email", "@gmail")
// 生成：WHERE email LIKE '%@gmail%'
```

### WhereRaw - 原生条件

```go
// 原生 SQL 条件
db.Table("users").
    WhereRaw("age > ? AND (status = ? OR vip = ?)", 18, 1, 1)

// 日期函数
db.Table("orders").
    WhereRaw("DATE(create_time) = CURDATE()")
```

### WhereOr - OR 条件

```go
// OR 条件
db.Table("users").
    Where("status", 1).
    WhereOr("vip", 1)
// 生成：WHERE status = 1 OR vip = 1

// 原生 OR 条件
db.Table("users").
    WhereRaw("status = 1").
    WhereOrRaw("vip = 1")
```

### WhereMap - 批量条件

```go
// 批量设置条件
conditions := map[string]interface{}{
    "status": 1,
    "city":   "北京",
    "level":  2,
}

db.Table("users").
    WhereMap(conditions)
```

## 存在性检查

### Exists - 判断记录是否存在

```go
exists, err := db.Table("users").
    Where("email", "test@example.com").
    Exists()

if err != nil {
    log.Fatal(err)
}

if exists {
    fmt.Println("用户已存在")
} else {
    fmt.Println("用户不存在")
}
```

### DoesntExist - 判断记录是否不存在

```go
notExists, err := db.Table("users").
    Where("email", "test@example.com").
    DoesntExist()

if notExists {
    fmt.Println("用户不存在")
}
```

## 去重查询

### Distinct - 去重

```go
// 查询不重复的城市
cities, err := db.Table("users").
    Distinct().
    Pluck("city")
```

## 锁定查询

### LockForUpdate - 排他锁

```go
// 排他锁 FOR UPDATE（用于更新）
var user User
err := db.Table("users").
    Where("id", 1).
    LockForUpdate().
    Scan(&user)
```

### Lock - 共享锁

```go
// 共享锁 LOCK IN SHARE MODE（用于读取）
var user User
err = db.Table("users").
    Where("id", 1).
    Lock("LOCK IN SHARE MODE").
    Scan(&user)
```

## 完整示例

### 复杂查询

```go
var users []User
err := db.Table("users u").
    LeftJoin("user_profiles up", "u.id = up.user_id").
    Select("u.id", "u.name", "u.email", "up.bio", "up.avatar").
    Where("u.status", 1).
    WhereIn("u.level", []int{1, 2, 3}).
    WhereNotNull("u.email").
    WhereLike("u.name", "张").
    OrderByDesc("u.create_time").
    Limit(20).
    Find(&users)
```

### 高级查询示例

```go
// 使用分页
result, err := db.Table("users").
    Where("status", 1).
    WhereBetween("age", 18, 30).
    Paginate(1, 20)

if err != nil {
    log.Fatal(err)
}

fmt.Printf("总数：%d, 当前页：%d, 总页数：%d\n", 
    result.Total, result.CurrentPage, result.LastPage)

// 使用分块处理大数据集
err = db.Table("users").
    Where("status", 1).
    Chunk(100, func(rows []map[string]interface{}) error {
        for _, row := range rows {
            fmt.Printf("处理用户：%v\n", row["name"])
        }
        return nil
    })
```

## 注意事项

1. `Scan` 和 `Find` 将结果填充到结构体
2. `Get` 和 `First` 返回 `map[string]interface{}`
3. 条件方法会累积形成 AND 条件
4. 使用 `WhereOr` 或 `WhereOrRaw` 添加 OR 条件
5. 所有查询自动路由到从库（读写分离）
6. 使用预处理语句，自动防止 SQL 注入
7. 使用 `Debug()` 可以打印 SQL 进行调试

## 调试功能

### Debug - 调试模式

```go
// 开启调试模式，会打印 SQL 和参数
var users []User
db.Debug().
    Table("users").
    Where("status", 1).
    Find(&users)
```

### Dump - 打印 SQL

```go
// 打印最后执行的 SQL
db.Table("users").
    Where("id", 1).
    Dump()
// 输出:
// SQL: SELECT * FROM users WHERE id = ?
// Args: [1]
```

### ToSQL - 获取 SQL（不执行）

```go
sql := db.Table("users").
    Where("id", 1).
    ToSQL()

fmt.Println("SQL:", sql)
```
