# 查询操作详解

## 简介
框架提供丰富的查询方法，支持单条查询、多条查询、条件查询、聚合查询等。

## 单条查询

### Find - 查询单条记录
```go
// 根据ID查询
user, err := db.Table("users").Where("id", 1).Find()

// 根据条件查询
user, err := db.Table("users").
    Where("email", "test@example.com").
    Find()

// 检查结果
if err != nil {
    log.Fatal(err)
}
if user == nil {
    fmt.Println("记录不存在")
} else {
    fmt.Printf("用户名: %v\n", user["name"])
    fmt.Printf("邮箱: %v\n", user["email"])
}
```

### Find 返回值
- 成功时返回 `map[string]interface{}`，包含查询结果
- 未找到记录时返回 `nil`，不是错误
- 查询出错时返回 `error`

## 多条查询

### Select - 查询多条记录
```go
// 查询所有记录
users, err := db.Table("users").Select()

// 条件查询
users, err := db.Table("users").
    Where("status", 1).
    Order("create_time DESC").
    Select()

// 遍历结果
for _, user := range users {
    fmt.Printf("ID: %v, 姓名: %v\n", user["id"], user["name"])
}
```

### Select 返回值
- 返回 `[]map[string]interface{}` 切片
- 没有记录时返回空切片，不是 `nil`

## 字段查询

### Value - 获取单个字段值
```go
// 获取单个字段
name, err := db.Table("users").Where("id", 1).Value("name")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("用户名: %v\n", name)
```

### Column - 获取列值列表
```go
// 获取所有用户邮箱
emails, err := db.Table("users").
    Where("status", 1).
    Column("email")

if err != nil {
    log.Fatal(err)
}

for _, email := range emails {
    fmt.Printf("邮箱: %v\n", email)
}
```

## 条件查询

### Where - 基本条件
```go
// 等于
db.Table("users").Where("status", 1)

// 大于
db.Table("users").Where("age", ">", 18)

// 大于等于
db.Table("users").Where("age", ">=", 18)

// 小于
db.Table("users").Where("age", "<", 65)

// 小于等于
db.Table("users").Where("age", "<=", 65)

// 不等于
db.Table("users").Where("status", "<>", 0)
db.Table("users").WhereNot("status", 0)

// 多条件组合
users, err := db.Table("users").
    Where("status", 1).
    Where("age", ">=", 18).
    Where("city", "北京").
    Select()
```

### WhereIn - IN 条件
```go
// IN 条件
users, err := db.Table("users").
    WhereIn("id", []interface{}{1, 2, 3, 4, 5}).
    Select()

// NOT IN 条件
users, err := db.Table("users").
    WhereNotIn("status", []interface{}{0, -1}).
    Select()
```

### WhereBetween - 范围条件
```go
// BETWEEN 条件
users, err := db.Table("users").
    WhereBetween("age", 18, 65).
    Select()

// NOT BETWEEN 条件
users, err := db.Table("users").
    WhereNotBetween("age", 0, 17).
    Select()

// 日期范围
orders, err := db.Table("orders").
    WhereBetween("create_time", "2024-01-01", "2024-12-31").
    Select()
```

### WhereNull - 空值条件
```go
// IS NULL
users, err := db.Table("users").
    WhereNull("delete_time").
    Select()

// IS NOT NULL
users, err := db.Table("users").
    WhereNotNull("email").
    Select()
```

### WhereLike - 模糊查询
```go
// LIKE 条件（自动添加 %%）
users, err := db.Table("users").
    WhereLike("name", "张").
    Select()
// 生成: WHERE name LIKE '%张%'

// 邮箱域名查询
users, err := db.Table("users").
    WhereLike("email", "@gmail").
    Select()
// 生成: WHERE email LIKE '%@gmail%'
```

### WhereRaw - 原生条件
```go
// 原生 SQL 条件
users, err := db.Table("users").
    WhereRaw("age > ? AND (status = ? OR vip = ?)", 18, 1, 1).
    Select()

// 日期函数
orders, err := db.Table("orders").
    WhereRaw("DATE(create_time) = CURDATE()").
    Select()
```

### WhereOr - OR 条件
```go
// OR 条件
users, err := db.Table("users").
    Where("status", 1).
    WhereOr("vip", 1).
    Select()
// 生成: WHERE status = 1 OR vip = 1

// 原生 OR 条件
users, err := db.Table("users").
    WhereRaw("status = 1").
    WhereOrRaw("vip = 1").
    Select()
```

### WhereMap - 批量条件
```go
// 批量设置条件
conditions := map[string]interface{}{
    "status": 1,
    "city":   "北京",
    "level":  2,
}

users, err := db.Table("users").
    WhereMap(conditions).
    Select()
```

## 存在性检查

### Exists - 判断记录是否存在
```go
exists, err := db.Table("users").
    Where("email", "test@example.com").
    Exists()

if err != nil {
    log.Fatal(err)
}

if exists {
    fmt.Println("用户已存在")
} else {
    fmt.Println("用户不存在")
}
```

## 去重查询

### Distinct - 去重
```go
// 查询不重复的城市
cities, err := db.Table("users").
    Field("DISTINCT city").
    Select()
```

## 锁定查询

### Lock - 查询锁定
```go
// 排他锁 FOR UPDATE（用于更新）
user, err := db.Table("users").
    Where("id", 1).
    Lock().
    Find()

// 共享锁 LOCK IN SHARE MODE（用于读取）
user, err := db.Table("users").
    Where("id", 1).
    Lock("LOCK IN SHARE MODE").
    Find()
```

## 完整示例

### 复杂查询
```go
users, err := db.Table("users u").
    Join("user_profiles up", "u.id = up.user_id", "LEFT").
    Field("u.id, u.name, u.email, up.bio, up.avatar").
    Where("u.status", 1).
    WhereIn("u.level", []interface{}{1, 2, 3}).
    WhereNotNull("u.email").
    WhereLike("u.name", "张").
    Order("u.create_time DESC").
    Limit(20).
    Select()
```

## 注意事项
1. `Find` 返回 `nil` 表示未找到记录，不是错误
2. `Select` 返回空切片表示没有记录
3. 条件方法会累积形成 AND 条件
4. 使用 `WhereOr` 或 `WhereOrRaw` 添加 OR 条件
5. 所有查询自动路由到从库（读写分离）
6. 使用预处理语句，自动防止SQL注入
