# 助手函数使用详解

Vigo 框架提供丰富的助手函数（Helper Functions），类似 ThinkPHP 和 Laravel 的全局辅助函数。这些函数可以在全局任意位置调用，无需导入复杂的依赖，让代码更加简洁高效。

---

## 一、什么是助手函数

### 1.1 助手函数概念

助手函数是框架提供的全局工具函数，用于：

- **简化操作**: 将复杂操作封装为简单函数
- **提高效率**: 减少重复代码
- **统一标准**: 提供一致的实现方式
- **全局可用**: 在任何位置都可以调用

### 1.2 使用方式

```go
import "vigo/framework/helper"

// 字符串处理
name := helper.Str("hello")
length := helper.Strlen("你好")

// 加密哈希
hash := helper.Md5("password")

// 时间日期
timestamp := helper.Time()
dateStr := helper.Date("Y-m-d H:i:s")

// 数组操作
exists := helper.InArray("value", array)
```

## 二、字符串处理函数

### 2.1 类型转换

```go
import "vigo/framework/helper"

// 转换为字符串
str := helper.Str(123)           // "123"
str := helper.Str(12.34)         // "12.34"
str := helper.Str(true)          // "true"
str := helper.Str(nil)           // ""

// 转换为整数
num := helper.Int("123")         // 123
num := helper.Int(12.99)         // 12
num := helper.Int(true)          // 1

// 转换为 int64
num64 := helper.Int64("123")     // 123

// 转换为浮点数
float := helper.Float("12.34")   // 12.34

// 转换为布尔值
bool := helper.Bool("true")      // true
bool := helper.Bool("1")         // true
bool := helper.Bool("yes")       // true
bool := helper.Bool(1)           // true
```

### 2.2 字符串操作

```go
// 截取字符串
str := helper.Substr("Hello World", 0, 5)    // "Hello"
str := helper.Substr("你好世界", 0, 2)         // "你好"

// 获取字符串长度
length := helper.Strlen("Hello")             // 5
length := helper.Strlen("你好")              // 2

// 查找字符串位置
pos := helper.Strpos("hello world", "world") // 6

// 替换字符串
str := helper.StrReplace("world", "Go", "hello world")  // "hello Go"

// 字符串分割
arr := helper.Explode(",", "a,b,c")          // ["a", "b", "c"]

// 数组转字符串
str := helper.Implode(",", []string{"a", "b", "c"})  // "a,b,c"

// 首字母大写
str := helper.Ucfirst("hello")               // "Hello"

// 首字母小写
str := helper.Lcfirst("Hello")               // "hello"

// 每个单词首字母大写
str := helper.Ucwords("hello world")         // "Hello World"

// 去除空白字符
str := helper.Trim("  hello  ")              // "hello"

// HTML 转义
str := helper.Htmlentities("<script>")       // "&lt;script&gt;"

// HTML 转义反转义
str := helper.HtmlEntityDecode("&lt;script&gt;")  // "<script>"

// 换行符转<br>
str := helper.Nl2br("line1\nline2")          // "line1<br>line2"

// 去除 HTML 标签
str := helper.StripTags("<p>Hello</p>")      // "Hello"
```

## 三、加密哈希函数

### 3.1 哈希加密

```go
// MD5 加密
hash := helper.Md5("password")
// 输出：5f4dcc3b5aa765d61d8327deb882cf99

// SHA1 加密
hash := helper.Sha1("password")
// 输出：5baa61e4c9b93f3f0682250b6cf8331b7ee68fd8

// SHA256 加密
hash := helper.Sha256("password")
// 输出：5e884898da28047151d0e56f8dc6292773603d0d6aabbdd62a11ef721d1542d8
```

### 3.2 Base64 编解码

```go
// Base64 编码
encoded := helper.Base64Encode("Hello World")
// 输出：SGVsbG8gV29ybGQ=

// Base64 解码
decoded, err := helper.Base64Decode("SGVsbG8gV29ybGQ=")
// 输出：Hello World
```

### 3.3 密码处理

```go
// 密码哈希
hash, err := helper.PasswordHash("password")

// 密码验证
valid := helper.PasswordVerify("password", hash)
```

## 五、时间日期函数

### 5.1 时间戳

```go
// 获取当前时间戳（秒）
ts := helper.Time()              // 例如：1709251200

// 获取当前时间戳（毫秒）
ts := helper.Millisecond()       // 例如：1709251200123

// 获取当前时间戳（微秒）
ts := helper.Microsecond()       // 例如：1709251200123456
```

### 5.2 日期格式化

```go
// 格式化时间戳
date := helper.Date("Y-m-d H:i:s")                    // "2024-03-01 12:00:00"
date := helper.Date("Y-m-d")                          // "2024-03-01"
date := helper.Date("H:i:s")                          // "12:00:00"

// 格式化指定时间戳
date := helper.Date("Y-m-d", 1709251200)              // "2024-03-01"

// 获取当前日期时间
datetime := helper.Datetime()                         // "2024-03-01 12:00:00"

// 获取今天日期
today := helper.Today()                               // "2024-03-01"

// 获取明天日期
tomorrow := helper.Tomorrow()                         // "2024-03-02"

// 获取昨天日期
yesterday := helper.Yesterday()                       // "2024-02-29"
```

### 5.3 解析时间字符串

```go
// 解析时间字符串为时间戳
ts := helper.Strtotime("2024-03-01 12:00:00")
```

## 十、完整示例

### 10.1 用户注册示例

```go
package controller

import (
    "vigo/framework/mvc"
    "vigo/framework/helper"
    "vigo/framework/facade"
)

// Register 用户注册
func Register(c *mvc.Context) {
    // 获取参数
    username := helper.Trim(c.PostForm("username", ""))
    password := c.PostForm("password", "")
    phone := helper.Trim(c.PostForm("phone", ""))
    
    // 验证参数
    if helper.Empty(username) {
        c.Json(400, map[string]interface{}{
            "code":    400,
            "message": "用户名不能为空",
        })
        return
    }
    
    if helper.Empty(password) {
        c.Json(400, map[string]interface{}{
            "code":    400,
            "message": "密码不能为空",
        })
        return
    }
    
    // 验证手机号
    if !helper.Empty(phone) && len(helper.Str(phone)) != 11 {
        c.Json(400, map[string]interface{}{
            "code":    400,
            "message": "手机号格式错误",
        })
        return
    }
    
    // 密码加密
    passwordHash := helper.Md5(password + "_salt")
    
    // 生成用户 Token
    token := helper.Md5(helper.RandomString(32) + helper.Str(helper.Time()))
    
    // 保存用户
    userID := helper.Time() // 示例：使用时间戳作为用户 ID
    facade.Redis.Set("user:"+helper.Str(userID), map[string]interface{}{
        "username": username,
        "password": passwordHash,
        "phone":    helper.MaskPhone(phone),  // 手机号脱敏
    }, 0)
    
    // 保存 Token
    facade.Redis.Set("token:"+token, userID, 24*3600)
    
    c.Json(200, map[string]interface{}{
        "code":    0,
        "message": "注册成功",
        "data": map[string]interface{}{
            "user_id": userID,
            "token":   token,
        },
    })
}
```

func SendSmsCode(c *mvc.Context) {

## 十一、最佳实践

### 11.1 导入别名

```go
import (
    h "vigo/framework/helper"
)

// 使用别名简化调用
str := h.Str(123)
md5 := h.Md5("password")
```

### 11.2 组合使用

```go
// 链式调用
result := helper.Ucfirst(helper.Trim("  hello  "))
// 输出："Hello"

// 组合加密
token := helper.Md5(helper.Sha1(helper.RandomString(10)))
```

### 11.3 错误处理

```go
// Base64 解码需要处理错误
decoded, err := helper.Base64Decode(encoded)
if err != nil {
    log.Error("解码失败:", err)
    return
}

// JSON 解码需要处理错误
var data map[string]interface{}
if err := helper.JsonDecode(jsonStr, &data); err != nil {
    log.Error("JSON 解码失败:", err)
    return
}
```
