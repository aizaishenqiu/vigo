# Request 模块使用指南

## 概述

Request 模块提供简洁、安全的请求数据处理：

- ✅ **自动净化** - 所有数据自动 HTML 转义
- ✅ **类型转换** - 支持字符串、整数、浮点数、布尔值
- ✅ **便捷方法** - 一行代码获取任何参数
- ✅ **请求判断** - 轻松判断请求方法、Ajax 等
- ✅ **IP 获取** - 支持代理 IP 识别

**重要**：Request 模块**只负责数据获取和净化**，**安全验证由中间件统一处理**。

## 快速开始

### 基本用法

```go
package controller

import (
    "vigo/framework/request"
)

func (u *UserController) Create(c *mvc.Context) {
    // 获取参数（已自动净化）
    username := request.Input(c, "username", "")
    email := request.Input(c, "email", "")
    age := request.InputInt(c, "age", 0)
    
    // 直接使用，无需担心安全问题
    user := &User{
        Username: username,
        Email:    email,
        Age:      age,
    }
    
    // 保存到数据库...
}
```

## 完整示例

func (u *UserController) Register(c *mvc.Context) {
    // 获取参数
    username := request.Input(c, "username", "")
    email := request.Input(c, "email", "")
    password := request.Input(c, "password", "")
    age := request.InputInt(c, "age", 0)
    
    // 验证必填项
    if username == "" || email == "" || password == "" {
        c.JSON(400, map[string]interface{}{
            "code":    400,
            "message": "必填参数不能为空",
        })
        return
    }
    
    // 创建用户
    user := &User{
        Username: username,
        Email:    email,
        Password: password,
        Age:      age,
    }
    
    // 保存到数据库...
    
    c.JSON(200, map[string]interface{}{
        "code":    200,
        "message": "注册成功",
        "data":    user,
    })
}
```

```

## 最佳实践

### 1. 使用助手函数（推荐）

```go
// ✅ 推荐：简洁明了
username := request.Input(c, "username", "")
age := request.InputInt(c, "age", 0)

// ❌ 不推荐：过于冗长
req := request.R(c)
username := req.Get("username", "")
age := req.GetInt("age", 0)
```

### 2. 设置合理的默认值

```go
// ✅ 推荐：设置默认值
page := request.QueryInt(c, "page", 1)
pageSize := request.QueryInt(c, "page_size", 10)

// ❌ 不推荐：使用 0 或空值
page := request.QueryInt(c, "page", 0) // 可能导致分页错误
```

### 3. 验证必填参数

```go
func Create(c *mvc.Context) {
    username := request.Input(c, "username", "")
    email := request.Input(c, "email", "")
    
    if username == "" || email == "" {
        c.JSON(400, map[string]interface{}{
            "code":    400,
            "message": "必填参数不能为空",
        })
        return
    }
    
    // 处理业务...
}
```

### 4. 批量获取参数

```go
// ✅ 推荐：批量获取
data := request.Only(c, "username", "email", "age")

// ❌ 不推荐：逐个获取
username := request.Input(c, "username", "")
email := request.Input(c, "email", "")
age := request.InputInt(c, "age", 0)
```

### 5. 使用请求判断

```go
func Handle(c *mvc.Context) {
    if request.IsPost(c) {
        // 处理 POST 请求
    } else if request.IsGet(c) {
        // 处理 GET 请求
    }
}
```
