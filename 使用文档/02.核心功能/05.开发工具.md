# 🛠️ 开发工具与部署

## Air 热重载 (推荐)

开发过程中，修改代码无需手动重启。

1.  **安装 Air**:
    ```bash
    go install github.com/air-verse/air@latest
    ```
2.  **启动**:
    在项目根目录运行 `air`。

配置文件为 `.air.toml`。

## CMD 窗口控制 (Windows)

在 `config.yaml` 中配置 `app.console`：

```yaml
app:
  console: true # 设置为 false 则启动后自动隐藏 CMD 窗口
```

## 编译部署

### Windows

```powershell
go build -o vigo.exe
```

### Linux

```bash
CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build -o vigo
```

### 生产环境运行

1.  将编译好的二进制文件上传至服务器。
2.  确保 `config.yaml` 配置正确。
3.  运行：
    ```bash
    ./vigo
    ```
