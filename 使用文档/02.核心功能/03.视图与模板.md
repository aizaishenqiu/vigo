# 🎨 视图与模板

框架支持标准 HTML 模板渲染。

## 目录结构

视图文件存放于 `app/view/` 目录下。

```
app/view/
└── home/
    └── index.html
```

## 在控制器中渲染

使用 `c.View()` 方法：

```go
func (c *HomeController) Index(ctx *mvc.Context) {
    c.Init(ctx)
    
    // 模板变量
    data := map[string]interface{}{
        "Title": "首页",
        "User":  "Admin",
    }
    
    // 渲染 app/view/home/index.html
    c.View("home/index.html", data)
}
```

## 模板语法

使用 Go 标准模板语法 `{{ }}`。

```html
<!DOCTYPE html>
<html>
<body>
    <h1>{{.Title}}</h1>
    <p>Hello, {{.User}}</p>
</body>
</html>
```
