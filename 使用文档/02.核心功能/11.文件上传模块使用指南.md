# 文件上传模块使用指南

## 概述

文件上传模块提供安全、可靠的文件上传功能，包括：

- ✅ **文件类型验证** - 白名单机制
- ✅ **文件大小限制** - 防止超大文件攻击
- ✅ **MIME 类型检测** - 阻止危险文件
- ✅ **文件名安全处理** - 防止路径穿越
- ✅ **自动重命名** - 时间戳 + 哈希，避免冲突
- ✅ **文件哈希计算** - SHA256 完整性校验

## 快速开始

### 基本用法

```go
package controller

import (
    "net/http"
    "vigo/framework/mvc"
    "vigo/framework/upload"
)

type FileController struct {
    BaseController
}

func (f *FileController) Upload(c *mvc.Context) {
    f.Init(c)
    
    // 创建上传处理器
    uploader := upload.NewUploader(
        "./uploads",                    // 上传目录
        5*1024*1024,                    // 最大 5MB
        []string{".jpg", ".png", ".gif"}, // 允许的扩展名
    )
    
    // 验证并上传
    result, err := uploader.ValidateAndUpload(c, "file")
    if err != nil {
        c.JSON(http.StatusBadRequest, map[string]interface{}{
            "code":    400,
            "message": err.Error(),
            "success": false,
        })
        return
    }
    
    // 返回上传结果
    c.JSON(http.StatusOK, map[string]interface{}{
        "code":    200,
        "message": "上传成功",
        "success": true,
        "data":    result,
    })
}
```

## 上传结果

UploadResult 结构包含完整的上传信息：

```go
type UploadResult struct {
    Filename    string `json:"filename"`     // 原始文件名
    SavePath    string `json:"save_path"`    // 保存路径（绝对路径）
    URL         string `json:"url"`          // 访问 URL
    Size        int64  `json:"size"`         // 文件大小（字节）
    Hash        string `json:"hash"`         // SHA256 哈希值
    ContentType string `json:"content_type"` // MIME 类型
}
```

## 安全特性

### 1. 文件类型白名单

```go
// 只允许图片
allowedExts := []string{".jpg", ".jpeg", ".png", ".gif", ".webp"}

// 只允许文档
allowedExts := []string{".pdf", ".doc", ".docx", ".xls", ".xlsx"}

// 只允许压缩包
allowedExts := []string{".zip", ".rar", ".7z", ".tar", ".gz"}
```

### 2. 文件大小限制

```go
// 最大 2MB
maxSize := 2 * 1024 * 1024

// 最大 10MB
maxSize := 10 * 1024 * 1024

// 最大 100MB
maxSize := 100 * 1024 * 1024
```

### 3. 危险的 MIME 类型自动拦截

模块会自动阻止以下危险 MIME 类型：

- `application/x-msdownload` - Windows 可执行文件
- `application/x-executable` - Linux 可执行文件
- `text/x-php` - PHP 脚本
- `text/x-python` - Python 脚本
- `application/x-perl` - Perl 脚本
- `application/x-sh` - Shell 脚本

### 4. 文件名安全处理

- 自动移除路径穿越字符（`..`, `/`, `\`）
- 检查危险字符（`<`, `>`, `:`, `"`, `|`, `?`, `*`, `&`, `;`, `$`, `` ` ``）
- 文件名长度限制（255 字符）

### 5. 自动重命名

上传的文件会自动重命名为：`时间戳_哈希。扩展名`

例如：`20260302150405_a1b2c3d4e5f6.jpg`

**优点**：
- 避免文件名冲突
- 防止中文乱码
- 防止恶意文件名
- 便于文件管理

### 多图上传

```go
package controller

import (
    "net/http"
    "vigo/framework/mvc"
    "vigo/framework/upload"
)

type ProductController struct {
    BaseController
}

// UploadProductImages 上传商品图片
func (p *ProductController) UploadProductImages(c *mvc.Context) {
    p.Init(c)
    
    uploader := upload.NewUploader(
        "./uploads/products",
        5*1024*1024, // 每张最大 5MB
        []string{".jpg", ".jpeg", ".png", ".webp"},
    )
    
    results, err := uploader.UploadMultiple(c, "images")
    if err != nil {
        c.JSON(http.StatusBadRequest, map[string]interface{}{
            "code":    400,
            "message": "图片上传失败：" + err.Error(),
            "success": false,
        })
        return
    }
    
    // 限制最多 9 张图片
    if len(results) > 9 {
        c.JSON(http.StatusBadRequest, map[string]interface{}{
            "code":    400,
            "message": "最多上传 9 张图片",
            "success": false,
        })
        return
    }
    
    // 提取 URL 列表
    urls := make([]string, 0, len(results))
    for _, result := range results {
        urls = append(urls, result.URL)
    }
    
    c.JSON(http.StatusOK, map[string]interface{}{
        "code":    200,
        "message": "图片上传成功",
        "success": true,
        "data": map[string]interface{}{
            "count": len(results),
            "urls":  urls,
        },
    })
}
```

```go
// 定期清理上传目录
func CleanOldFiles() {
    filepath.Walk("./uploads", func(path string, info os.FileInfo, err error) error {
        if err != nil {
            return err
        }
        
        // 删除超过 30 天的文件
        if time.Since(info.ModTime()) > 30*24*time.Hour {
            os.Remove(path)
        }
        return nil
    })
}
```

## 注意事项

1. **上传目录权限**：确保 Web 服务器有写权限
2. **磁盘空间监控**：定期检查磁盘使用情况
3. **文件类型验证**：永远不要信任客户端的文件类型声明
4. **病毒扫描**：重要应用建议集成病毒扫描
5. **CDN 加速**：大文件建议使用 CDN 分发
