# 🎮 路由与控制器

## 路由定义

路由文件位于 `route/route.go`。

```go
package route

import (
    "vigo/app/controller"
    "vigo/framework/mvc"
)

func Init(r *mvc.Router) {
    // 实例化控制器
    index := &controller.IndexController{}
    
    // 注册 GET 请求
    r.GET("/", index.Index)
    r.GET("/hello", index.Hello)
    
    // 注册 POST 请求
    auth := &controller.AuthController{}
    r.POST("/api/login", auth.Login)
    
    // 注册标准 http.Handler (如 Swagger)
    // r.Handle("/path", handler)
}
```

## 控制器编写

控制器位于 `app/controller/` 目录下，需继承 `BaseController`。

```go
package controller

import "vigo/framework/mvc"

type UserController struct {
    BaseController
}

// 示例方法
func (c *UserController) Profile(ctx *mvc.Context) {
    c.Init(ctx) // 必须调用初始化
    
    // 获取参数
    uid := c.Input("uid")
    
    // 返回 JSON
    c.Success(map[string]string{"uid": uid})
}
```

## 请求与响应 (Context)

`mvc.Context` 封装了常用的 HTTP 操作。

### 获取参数
*   `c.Input("key")`: 获取 URL Query 或 Form Post 参数。

### 返回响应
*   `c.Json(code, data)`: 返回原始 JSON。
*   `c.Success(data)`: 返回标准成功结构 `{"code":0, "msg":"success", "data": ...}`。
*   `c.Error(code, msg)`: 返回标准错误结构 `{"code":401, "msg":"...", "data": nil}`。
*   `c.View(name, data)`: 渲染 HTML 模板 (需在 Controller 中调用)。
*   `c.Redirect(url)`: 重定向。
