# Request 使用指南

---

## 一、概述

Vigo 框架提供统一的 Request 模块，用于安全、便捷地处理 HTTP 请求数据。

### 核心特性

- ✅ **自动净化** - 所有数据自动 HTML 转义，防止 XSS 攻击
- ✅ **类型转换** - 支持字符串、整数、浮点数、布尔值、数组等
- ✅ **便捷方法** - 一行代码获取任何参数
- ✅ **请求判断** - 轻松判断请求方法、Ajax 等
- ✅ **IP 获取** - 支持代理 IP 识别
- ✅ **安全验证** - 自动检测 SQL 注入、XSS、命令注入等攻击

### 三种使用方式

#### 1. 自动验证（推荐）

通过中间件自动验证所有请求，无需手动调用：

```go
// 中间件自动验证所有请求
// 控制器中直接获取净化后的数据
username := request.Input(c, "username", "")
email := request.Input(c, "email", "")
```

#### 2. 助手函数（简洁优雅）

使用全局助手函数快速获取参数：

```go
// 获取字符串
username := request.Input(c, "username", "")
email := request.Input(c, "email", "")

// 获取整数
age := request.InputInt(c, "age", 0)
count := request.InputInt(c, "count", 10)

// 获取浮点数
price := request.InputFloat(c, "price", 0.0)

// 获取布尔值
isActive := request.InputBool(c, "is_active", false)

// 获取数组
tags := request.InputSlice(c, "tags", []string{})
```

#### 3. 链式调用（面向对象）

使用 Request 对象进行链式操作：

```go
req := request.R(c).
    Rule("username", validate.Required(), validate.MinLength(3)).
    Rule("email", validate.Required(), validate.Email()).
    Rule("age", validate.Min(18))

if !req.Validate() {
    c.JSON(map[string]interface{}{
        "code": 400,
        "msg":  "验证失败",
        "errors": req.GetErrors(),
    })
    return
}

username := req.Get("username", "")
email := req.Get("email", "")
age := req.GetInt("age", 0)
```

## 十一、完整示例

### 11.1 用户注册

```go
package controller

import (
    "vigo/framework/mvc"
    "vigo/framework/request"
    "vigo/framework/validate"
)

func (u *UserController) Register(c *mvc.Context) {
    // 创建验证器
    req := request.R(c).
        Rule("username", validate.Required(), validate.MinLength(3), validate.MaxLength(20)).
        Rule("email", validate.Required(), validate.Email()).
        Rule("password", validate.Required(), validate.MinLength(6)).
        Rule("age", validate.Min(18), validate.Max(100))

    // 执行验证
    if !req.Validate() {
        c.JSON(map[string]interface{}{
            "code":   400,
            "msg":    "验证失败",
            "errors": req.GetErrors(),
        })
        return
    }

    // 获取验证后的数据
    username := req.Get("username", "")
    email := req.Get("email", "")
    password := req.Get("password", "")
    age := req.GetInt("age", 0)

    // 创建用户
    user := &User{
        Username: username,
        Email:    email,
        Password: password,
        Age:      age,
    }

    // 保存到数据库...

    c.JSON(map[string]interface{}{
        "code": 200,
        "msg":  "注册成功",
        "data": user,
    })
}
```

func (a *ArticleController) Search(c *mvc.Context) {
    // 获取查询参数
    keyword := request.Query(c, "keyword", "")
    page := request.QueryInt(c, "page", 1)
    limit := request.QueryInt(c, "limit", 20)
    category := request.QueryInt(c, "category", 0)
    sort := request.Query(c, "sort", "created_at")

    // 构建查询
    query := db.Table("articles")
    if keyword != "" {
        query.Where("title LIKE ? OR content LIKE ?", "%"+keyword+"%", "%"+keyword+"%")
    }
    if category > 0 {
        query.Where("category_id = ?", category)
    }
    query.Order(sort + " DESC")
    query.Page(page, limit)

        "code": 200,
        "data": articles,
    })
}

### 11.3 文件上传

```go
func (f *FileController) Upload(c *mvc.Context) {
    // 获取上传的文件
    file, err := c.FormFile("file")
    if err != nil {
        c.JSON(map[string]interface{}{
            "code": 400,
            "msg":  "请选择要上传的文件",
        })
        return
    }

    // 检查文件大小（最大 10MB）
    if file.Size > 10*1024*1024 {
        c.JSON(map[string]interface{}{
            "code": 400,
            "msg":  "文件大小不能超过 10MB",
        })
        return
    }

    // 检查文件类型
    allowedTypes := []string{".jpg", ".jpeg", ".png", ".gif"}
    ext := strings.ToLower(filepath.Ext(file.Filename))
    if !contains(allowedTypes, ext) {
        c.JSON(map[string]interface{}{
            "code": 400,
            "msg":  "不支持的文件类型",
        })
        return
    }

    // 保存到指定目录
    dst := filepath.Join("uploads", file.Filename)
    if err := c.SaveUploadedFile(file, dst); err != nil {
        c.JSON(map[string]interface{}{
            "code": 500,
            "msg":  "上传失败",
        })
        return
    }

    c.JSON(map[string]interface{}{
        "code": 200,
        "msg":  "上传成功",
        "data": map[string]string{
            "url": "/" + dst,
        },
    })
}
```

## 十三、最佳实践

### ✅ 推荐做法

1. **使用助手函数**

```go
// 简洁明了
username := request.Input(c, "username", "")
age := request.InputInt(c, "age", 0)
```

2. **设置合理的默认值**

```go
// 避免空指针
page := request.QueryInt(c, "page", 1)
limit := request.QueryInt(c, "limit", 20)
status := request.Input(c, "status", "active")
```

3. **验证必填参数**

```go
// 使用验证器
req := request.R(c).
    Rule("username", validate.Required()).
    Rule("email", validate.Required(), validate.Email())

if !req.Validate() {
    // 处理错误
}
```

4. **批量获取参数**

```go
// 只获取需要的字段
data := request.Only(c, "username", "email", "age")

// 排除敏感字段
data := request.Except(c, "password", "token")
```

5. **使用请求判断**

```go
// 根据请求方法返回不同响应
if request.IsAjax(c) {
    c.JSON(data)
} else {
    c.HTML("template", data)
}
```

### ❌ 避免做法

1. **不要直接使用原始数据**

```go
// ❌ 避免
rawData := c.Request.FormValue("username")

// ✅ 推荐
username := request.Input(c, "username", "")
```

2. **不要忘记验证**

```go
// ❌ 避免
username := request.Input(c, "username", "")
// 直接使用 username...

// ✅ 推荐
req := request.R(c).Rule("username", validate.Required())
if !req.Validate() {
    return
}
username := req.Get("username", "")
```

3. **不要忽略错误**

```go
// ❌ 避免
data := request.JSON(c)
// 直接使用 data...

// ✅ 推荐
data := request.JSON(c)
if data == nil {
    c.JSON(map[string]interface{}{
        "code": 400,
        "msg":  "无效的 JSON 数据",
    })
    return
}
```

### Q3: 如何自定义验证规则？

```go
// 创建自定义验证规则
func CustomRule(value interface{}) bool {
    // 自定义验证逻辑
    return true
}

// 使用自定义规则
req := request.R(c).
    Rule("field", validate.Required(), CustomRule)
```

### Q4: 验证失败会返回什么？

验证失败时，可以通过 `GetErrors()` 获取错误信息：

```go
if !req.Validate() {
    errors := req.GetErrors()
    // errors 格式：
    // map[string][]string{
    //     "username": {"用户名不能为空", "用户名长度至少为 3"},
    //     "email": {"邮箱格式不正确"},
    // }
}
```
