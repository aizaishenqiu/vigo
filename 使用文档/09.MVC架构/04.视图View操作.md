# 视图（View）操作

## 简介
视图是MVC架构中的展示层，负责将数据渲染成用户界面。框架使用Go标准库的html/template引擎进行模板渲染。

## 视图引擎

### 创建视图引擎
```go
import "vigo/framework/view"

// 创建模板引擎
engine := view.NewTemplateEngine("app/view")

// 设置模板后缀（默认.html）
engine.SetSuffix(".html")

// 启用模板缓存（生产环境推荐）
engine.EnableCache(true)
```

### 视图引擎配置
```go
type TemplateEngine struct {
    dir      string            // 模板目录
    suffix   string            // 模板后缀
    funcMap  template.FuncMap  // 自定义函数
    cache    map[string]*template.Template
    useCache bool              // 是否启用缓存
}
```

## 模板渲染

### 基本渲染
```go
// 在控制器中渲染模板
func (c *PageController) Home(ctx *mvc.Context) {
    ctx.HTML(200, "home.html", map[string]interface{}{
        "Title": "首页",
    })
}
```

### 渲染用户列表
```go
func (c *PageController) UserList(ctx *mvc.Context) {
    page := ctx.InputInt("page", 1)
    
    userModel := &model.UserModel{}
    users, total, _ := userModel.GetList(page, 10)
    
    ctx.HTML(200, "user/list.html", map[string]interface{}{
        "Title": "用户列表",
        "Users": users,
        "Total": total,
        "Page":  page,
    })
}
```

## 模板语法

### 变量输出
```html
<!-- 输出变量 -->
<p>用户名: {{.User.Name}}</p>
<p>邮箱: {{.User.Email}}</p>

<!-- 输出嵌套字段 -->
<p>城市: {{.User.Profile.City}}</p>
```

### 条件判断
```html
<!-- if/else -->
{{if .User}}
    <p>欢迎, {{.User.Name}}</p>
{{else}}
    <p>请先登录</p>
{{end}}

<!-- 比较判断 -->
{{if eq .User.Status 1}}
    <span class="badge">激活</span>
{{else}}
    <span class="badge">禁用</span>
{{end}}

<!-- 多条件判断 -->
{{if gt .User.Age 18}}
    <span>成年人</span>
{{else if gt .User.Age 12}}
    <span>青少年</span>
{{else}}
    <span>儿童</span>
{{end}}

<!-- 逻辑运算 -->
{{if and .User (gt .User.Age 18)}}
    <span>成年用户</span>
{{end}}

{{if or .User.Admin .User.SuperAdmin}}
    <span>管理员</span>
{{end}}
```

### 循环遍历
```html
<!-- 遍历列表 -->
<ul>
{{range .Users}}
    <li>
        <span>{{.Name}}</span>
        <span>{{.Email}}</span>
    </li>
{{end}}
</ul>

<!-- 带索引遍历 -->
<table>
{{range $index, $user := .Users}}
    <tr>
        <td>{{$index}}</td>
        <td>{{$user.Name}}</td>
        <td>{{$user.Email}}</td>
    </tr>
{{end}}
</table>

<!-- 遍历Map -->
{{range $key, $value := .Data}}
    <p>{{$key}}: {{$value}}</p>
{{end}}
```

### 模板嵌套
```html
<!-- 引入子模板 -->
{{template "partials/header.html" .}}

<!-- 定义区块 -->
{{define "content"}}
<div class="content">
    <h1>{{.Title}}</h1>
</div>
{{end}}

<!-- 使用区块 -->
{{block "content" .}}{{end}}
```

## 自定义模板函数

### 注册自定义函数
```go
import (
    "html/template"
    "time"
)

// 创建带自定义函数的引擎
func NewEngine() *view.TemplateEngine {
    engine := view.NewTemplateEngine("app/view")
    
    // 注册自定义函数
    engine.AddFunc("date", formatDate)
    engine.AddFunc("datetime", formatDatetime)
    engine.AddFunc("truncate", truncate)
    engine.AddFunc("safe", safeHTML)
    
    return engine
}

// 日期格式化
func formatDate(t time.Time) string {
    return t.Format("2006-01-02")
}

// 日期时间格式化
func formatDatetime(t time.Time) string {
    return t.Format("2006-01-02 15:04:05")
}

// 字符串截断
func truncate(s string, length int) string {
    if len(s) <= length {
        return s
    }
    return s[:length] + "..."
}

// 安全HTML输出
func safeHTML(s string) template.HTML {
    return template.HTML(s)
}
```

### 使用自定义函数
```html
<!-- 日期格式化 -->
<p>创建时间: {{.User.CreateTime | datetime}}</p>

<!-- 字符串截断 -->
<p>简介: {{.User.Bio | truncate 100}}</p>

<!-- 安全HTML -->
<div>{{.Content | safe}}</div>
```

## 完整页面示例

### 用户列表页面
```html
<!-- views/user/list.html -->
{{define "content"}}
<div class="user-list">
    <h1>用户列表</h1>
    
    <div class="actions">
        <a href="/users/create" class="btn btn-primary">添加用户</a>
    </div>
    
    <table class="table">
        <thead>
            <tr>
                <th>ID</th>
                <th>姓名</th>
                <th>邮箱</th>
                <th>状态</th>
                <th>创建时间</th>
                <th>操作</th>
            </tr>
        </thead>
        <tbody>
        {{range .Users}}
            <tr>
                <td>{{.id}}</td>
                <td>{{.name}}</td>
                <td>{{.email}}</td>
                <td>
                    {{if eq .status 1}}
                        <span class="badge badge-success">激活</span>
                    {{else}}
                        <span class="badge badge-danger">禁用</span>
                    {{end}}
                </td>
                <td>{{.create_time | datetime}}</td>
                <td>
                    <a href="/users/{{.id}}" class="btn btn-sm btn-info">详情</a>
                    <a href="/users/{{.id}}/edit" class="btn btn-sm btn-warning">编辑</a>
                    <button class="btn btn-sm btn-danger" onclick="deleteUser({{.id}})">删除</button>
                </td>
            </tr>
        {{end}}
        </tbody>
    </table>
    
    <div class="pagination">
        {{if gt .Page 1}}
            <a href="?page={{sub .Page 1}}" class="btn">上一页</a>
        {{end}}
        
        <span>第 {{.Page}} 页</span>
        
        <a href="?page={{add .Page 1}}" class="btn">下一页</a>
    </div>
</div>
{{end}}

{{define "scripts"}}
<script>
function deleteUser(id) {
    if (confirm('确定要删除这个用户吗？')) {
        fetch('/api/users/' + id, {
            method: 'DELETE',
            headers: {
                'Content-Type': 'application/json'
            }
        })
        .then(response => response.json())
        .then(data => {
            if (data.code === 0) {
                location.reload();
            } else {
                alert(data.msg);
            }
        });
    }
}
</script>
{{end}}
```

## 静态资源

### 配置静态资源目录
```go
// 在路由中配置静态资源
router.Handle("/static/", http.StripPrefix("/static/", 
    http.FileServer(http.Dir("public"))))
```

### 引用静态资源
```html
<!-- CSS -->
<link rel="stylesheet" href="/static/css/style.css">

<!-- JavaScript -->
<script src="/static/js/app.js"></script>

<!-- 图片 -->
<img src="/static/images/logo.png" alt="Logo">
```

## 注意事项
1. 模板文件统一放在 `app/view` 目录下
2. 使用布局模板减少重复代码
3. 注意XSS防护，默认会对HTML进行转义
4. 生产环境启用模板缓存提升性能
5. 复杂展示逻辑通过自定义函数实现
6. 静态资源建议使用CDN或独立服务器托管
