# 🏢 多租户 (Tenant)

SaaS 系统的核心组件，用于处理多租户上下文。

## 租户上下文

在中间件中解析并注入租户 ID：

```go
func TenantMiddleware() mvc.HandlerFunc {
    return func(c *mvc.Context) {
        // 从 Header 或 Token 解析
        tenantId := c.Request.Header.Get("X-Tenant-ID")
        if tenantId != "" {
            ctx := tenant.ContextWithTenant(c.Request.Context(), tenantId)
            c.Request = c.Request.WithContext(ctx)
        }
        c.Next()
    }
}
```

## 获取当前租户

```go
func (c *UserController) Index(ctx *mvc.Context) {
    tid, err := tenant.TenantFromContext(ctx.Request.Context())
    if err == nil {
        fmt.Println("Current Tenant:", tid)
    }
}
```
