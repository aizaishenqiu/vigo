# 🐇 RabbitMQ 消息队列

提供 AMQP 协议的完整封装，支持自动重连、管理界面。

## 配置

```yaml
rabbitmq:
  enabled: true
  host: 127.0.0.1
  port: 5672
  user: guest
  pass: guest
  vhost: /
  
  admin:
    enabled: true
    username: "admin"      # 管理界面用户名
    password: "your_password" # 管理界面密码
    port: 15672
```

## 发布消息

```go
import "vigo/framework/rabbitmq"

client := rabbitmq.New(rabbitmq.Config{
    Host: "localhost",
    Port: 5672,
    User: "guest",
    Password: "guest",
    Vhost: "/",
})

// 连接
client.Connect()

// 发送到队列
err := client.PublishToQueue("my_queue", []byte("Hello"))

// 通过交换机路由
err := client.Publish("logs_exchange", "error", []byte("Something went wrong"))
```

## 消费消息

```go
client.Consume("error_logs_queue", func(msg []byte) {
    fmt.Printf("Received: %s\n", msg)
})
```

## 队列管理

```go
// 声明队列
client.DeclareQueue("my_queue", true)  // 持久化

// 声明交换机
client.DeclareExchange("logs", "topic", true)

// 绑定队列到交换机
client.BindQueue("my_queue", "logs", "error.*")
```

## 状态检查

```go
client.IsConnected()  // bool
client.GetStatus()    // map
```

## 管理界面

启用 `admin.enabled: true` 后，访问 `/admin/rabbitmq` 进行可视化管理。
