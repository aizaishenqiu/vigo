# 🌐 Gateway (网关)

**Vigo** 提供了基础的网关组件，支持反向代理和路由转发。

## 使用 Proxy

`framework/gateway` 包中的 `Proxy` 方法可以快速创建一个反向代理处理器。

```go
import (
    "vigo/framework/gateway"
    "vigo/framework/mvc"
)

func Init(r *mvc.Router) {
    // 将 /api/user/* 的请求转发到 http://user-service:8080
    r.Handle("/api/user/", gateway.Proxy("http://user-service:8080"))
}
```

这对于构建微服务网关或前后端分离开发（解决跨域）非常有用。
