# 🔌 WebSocket

基于 Gorilla WebSocket 的封装，支持简单的连接管理和广播。

## 启动服务

在路由中注册 WebSocket 处理函数：

```go
import "vigo/framework/websocket"

func Init(r *mvc.Router) {
    hub := websocket.NewHub()
    go hub.Run()
    
    r.GET("/ws", func(c *mvc.Context) {
        websocket.Handler(hub, c.Writer, c.Request)
    })
}
```

## 广播消息

```go
hub.Broadcast <- []byte("Hello Everyone!")
```
