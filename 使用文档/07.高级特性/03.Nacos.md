# ☁️ Nacos 配置中心

框架提供了 Nacos 客户端的完整封装，支持配置管理、服务注册与发现、自动心跳。

## 配置

```yaml
nacos:
  host: 127.0.0.1
  port: 8848
  namespace: ""
  data_id: vigo-config
  group: DEFAULT_GROUP
  
  # 服务发现
  discovery:
    enabled: true
    service_name: vigo
    auto_register: true
    heartbeat: 5
    
  # 管理界面
  admin:
    enabled: true
    username: "admin"      # 管理界面用户名
    password: "your_password" # 管理界面密码
    port: 8848
```

## 基本使用

```go
import "vigo/framework/nacos"

cfg := nacos.Config{
    IpAddr:      "127.0.0.1",
    Port:        8848,
    NamespaceId: "public",
    DataId:      "vigo-config",
    Group:       "DEFAULT_GROUP",
}

client := nacos.NewClient(cfg)

// 获取配置
configStr, err := client.GetConfig()

// 发布配置
err := client.PublishConfig("app-config", "DEFAULT_GROUP", `{"key": "value"}`)
```

## 服务注册与发现

### 自动注册（推荐）

配置 `discovery.auto_register: true` 后，应用启动时自动注册并保持心跳。

### 手动注册

```go
// 注册服务
client.RegisterInstance("192.168.1.100", 8080, "user-service")

// 启动心跳
client.StartHeartbeat(5 * time.Second)

// 注销服务
client.DeregisterInstance("192.168.1.100", 8080, "user-service")
```

### 服务发现

```go
// 发现服务实例
instances, err := client.DiscoverService("user-service")
for _, inst := range instances {
    if inst.Healthy {
        fmt.Printf("实例: %s:%d\n", inst.IP, inst.Port)
    }
}
```

## Instance 结构

```go
type Instance struct {
    IP      string
    Port    int
    Healthy bool
    Weight  float64
}
```

## 通过门面访问

```go
import "vigo/framework/facade"

nc := facade.Nacos()
instances, _ := nc.DiscoverService("my-service")
```
