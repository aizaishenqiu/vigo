# 容器与依赖注入

**Vigo** 核心采用了一个轻量级的 IoC (控制反转) 容器，用于管理类的依赖关系。

## 概念

容器（Container）是现代框架的核心，它负责实例化对象并自动处理依赖。

## 使用容器

`framework/container/container.go` 提供了基础实现。

### 绑定 (Bind)

将一个接口或别名绑定到具体的实现（或构造函数）。

```go
import "vigo/framework/app"

// 绑定
app.Instance.Bind("my_service", func() interface{} {
    return &MyService{}
})
```

### 单例 (Singleton)

绑定一个单例，整个生命周期内只实例化一次。

```go
app.Instance.Singleton("db", dbInstance)
```

### 解析 (Make)

从容器中获取实例。

```go
service := app.Instance.Make("my_service").(*MyService)
```

## 应用实例 (App)

`App` 本身也是一个容器，并且是全局唯一的单例。

```go
import "vigo/framework/app"

// 获取全局 App
var application = app.Instance
```
