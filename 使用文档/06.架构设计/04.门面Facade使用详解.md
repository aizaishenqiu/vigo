# 门面（Facade）使用详解

Vigo 框架提供强大的门面（Facade）系统，灵感来自 Laravel 的门面模式。门面提供了一种优雅、简洁的方式来访问框架的核心功能，无需复杂的依赖注入。

---

## 一、什么是门面

### 1.1 门面概念

门面（Facade）是一种设计模式，它为复杂的子系统提供一个统一的、简化的接口。在 Vigo 框架中，门面让你可以：

- **静态调用**: 使用类似静态方法的方式调用实例方法
- **简洁语法**: 避免冗长的依赖注入代码
- **全局访问**: 在任意位置访问框架服务
- **易于测试**: 支持门面替换和 Mock

### 1.2 对比示例

**不使用门面（传统方式）：**
```go
// 需要手动获取实例
redis := container.Get("redis").(*redis.Client)
data := redis.Get("key")

db := container.Get("db").(*gorm.DB)
user := db.First(&User{}, 1)
```

**使用门面（简洁优雅）：**
```go
// 直接使用门面
data := Redis.Get("key")
user := DB.Table("users").First(1)
```

## 二、内置门面

Vigo 框架提供以下内置门面：

### 2.1 Redis 门面

```go
import "vigo/framework/facade"

// 字符串操作
facade.Redis.Set("name", "vigo", 0)
value := facade.Redis.Get("name")

// 哈希操作
facade.Redis.HSet("user:1", "name", "admin")
facade.Redis.HGet("user:1", "name")

// 列表操作
facade.Redis.LPush("queue", "item1")
facade.Redis.LPop("queue")

// 集合操作
facade.Redis.SAdd("tags", "go")
facade.Redis.SMembers("tags")

// 发布订阅
facade.Redis.Publish("channel", "message")
```

### 2.2 DB 门面

```go
import "vigo/framework/facade"

// 查询操作
users := facade.DB.Table("users").Where("status", 1).Get()
user := facade.DB.Table("users").Where("id", 1).First()

// 插入操作
facade.DB.Table("users").Insert(map[string]interface{}{
    "name": "admin",
    "email": "admin@example.com",
})

// 更新操作
facade.DB.Table("users").Where("id", 1).Update("name", "newname")

// 删除操作
facade.DB.Table("users").Where("id", 1).Delete()

// 事务操作
facade.DB.Transaction(func(tx *gorm.DB) error {
    tx.Table("users").Insert(...)
    tx.Table("orders").Insert(...)
    return nil
})
```

### 2.3 Cache 门面

```go
import "vigo/framework/facade"

// 缓存操作
facade.Cache.Set("key", "value", time.Hour)
value := facade.Cache.Get("key")

// 记住（缓存回调结果）
facade.Cache.Remember("user:1", time.Hour, func() interface{} {
    return DB.Table("users").First(1)
})

// 删除缓存
facade.Cache.Forget("key")
facade.Cache.Flush()  // 清空所有缓存
```

### 2.4 Log 门面

```go
import "vigo/framework/facade"

// 日志记录
facade.Log.Info("用户登录", "user_id", 1)
facade.Log.Error("数据库错误", "error", err)
facade.Log.Warning("配置缺失", "key", "api_key")
facade.Log.Debug("调试信息", "data", data)

// 不同通道
facade.Log.Channel("payment").Info("支付成功")
facade.Log.Channel("email").Error("发送失败")
```

### 2.5 Config 门面

```go
import "vigo/framework/facade"

// 读取配置
app := facade.Config.Get("app.name")
db := facade.Config.Get("database.host")
redis := facade.Config.Get("redis.port")

// 检查配置
if facade.Config.Has("app.debug") {
    // ...
}
```

### 2.6 Event 门面

```go
import "vigo/framework/facade"

// 触发事件
facade.Event.Dispatch("user.login", map[string]interface{}{
    "user_id": 1,
})

// 监听事件
facade.Event.Listen("user.login", func(data map[string]interface{}) {
    fmt.Println("用户登录:", data["user_id"])
})

// 异步触发
facade.Event.Async("user.register", data)
```

### 2.7 Queue 门面

```go
import "vigo/framework/facade"

// 推送任务到队列
facade.Queue.Push(&SendEmailJob{
    To: "user@example.com",
    Subject: "欢迎",
})

// 延迟执行
facade.Queue.Delay(5 * time.Minute).Push(&SendEmailJob{...})

// 指定队列
facade.Queue.On("emails").Push(&SendEmailJob{...})
```

## 三、自定义门面

### 3.1 创建门面类

```go
// app/facade/payment.go
package facade

import (
    "vigo/framework/extend"
    "vigo/app/service/payment"
)

// PaymentFacade 支付门面
type PaymentFacade struct {
    service *payment.PaymentService
}

// Name 返回门面名称
func (p *PaymentFacade) Name() string {
    return "payment"
}

// Boot 引导门面（应用启动时调用）
func (p *PaymentFacade) Boot() error {
    // 初始化服务
    p.service = payment.NewPaymentService()
    return nil
}

// GetAccessor 返回访问器
func (p *PaymentFacade) GetAccessor() interface{} {
    return p
}

// Pay 支付方法
func (p *PaymentFacade) Pay(orderID string, amount float64) error {
    return p.service.ProcessPayment(orderID, amount)
}

// Refund 退款方法
func (p *PaymentFacade) Refund(orderID string) error {
    return p.service.ProcessRefund(orderID)
}

// Query 查询订单
func (p *PaymentFacade) Query(orderID string) (*payment.Order, error) {
    return p.service.QueryOrder(orderID)
}
```

### 3.2 注册门面

```go
// app/facade/init.go
package facade

import (
    "vigo/framework/extend"
)

// Init 初始化所有门面
func Init() {
    // 注册支付门面
    extend.RegisterFacade(&PaymentFacade{})
    
    // 注册其他门面
    extend.RegisterFacade(&SmsFacade{})
    extend.RegisterFacade(&StorageFacade{})
}
```

### 3.3 使用自定义门面

```go
// app/controller/order.go
package controller

import (
    "vigo/framework/mvc"
    "vigo/app/facade"
)

// CreateOrder 创建订单
func CreateOrder(c *mvc.Context) {
    // 使用支付门面
    err := facade.Payment.Pay("order_123", 99.99)
    if err != nil {
        c.Json(500, map[string]interface{}{
            "code":    500,
            "message": "支付失败",
        })
        return
    }
    
    c.Json(200, map[string]interface{}{
        "code":    0,
        "message": "支付成功",
    })
}
```

## 四、门面管理器

### 4.1 手动注册门面

```go
import "vigo/framework/extend"

// 创建门面实例
myFacade := &MyFacade{}

// 注册门面
err := extend.RegisterFacade(myFacade)
if err != nil {
    log.Fatal("注册门面失败:", err)
}
```

### 4.2 获取门面实例

```go
import "vigo/framework/extend"

// 获取门面
facade := extend.GetFacade("payment")
if facade != nil {
    paymentFacade := facade.(*PaymentFacade)
    paymentFacade.Pay("order_123", 99.99)
}
```

### 4.3 检查门面是否存在

```go
if extend.HasFacade("payment") {
    // 门面存在
} else {
    // 门面不存在
}
```

## 五、最佳实践

### 5.1 命名规范

- 门面名称使用大驼峰命名（如：`PaymentFacade`）
- 门面实例使用简洁名称（如：`Payment`）
- 避免名称冲突

### 5.2 初始化时机

```go
// main.go
func main() {
    // 1. 初始化配置
    config.Init()
    
    // 2. 注册门面
    facade.Init()
    
    // 3. 引导门面
    extend.BootFacades()
    
    // 4. 启动应用
    app.Run()
}
```

### 5.3 错误处理

```go
type PaymentFacade struct {
    service *payment.PaymentService
}

func (p *PaymentFacade) Boot() error {
    // 初始化时处理错误
    var err error
    p.service, err = payment.NewPaymentService()
    if err != nil {
        return fmt.Errorf("初始化支付服务失败：%w", err)
    }
    return nil
}
```

### 5.4 依赖管理

```go
type OrderFacade struct {
    db      *gorm.DB
    redis   *redis.Client
    payment *PaymentFacade
}

func (o *OrderFacade) Boot() error {
    // 从容器获取依赖
    o.db = container.Get("db").(*gorm.DB)
    o.redis = container.Get("redis").(*redis.Client)
    o.payment = extend.GetFacade("payment").(*PaymentFacade)
    return nil
}
```

## 六、完整示例

### 6.1 短信门面

```go
// app/facade/sms.go
package facade

import (
    "vigo/framework/extend"
    "vigo/app/service/sms"
)

// SmsFacade 短信门面
type SmsFacade struct {
    service *sms.SmsService
}

// Name 返回门面名称
func (s *SmsFacade) Name() string {
    return "sms"
}

// Boot 引导门面
func (s *SmsFacade) Boot() error {
    s.service = sms.NewSmsService()
    return nil
}

// GetAccessor 返回访问器
func (s *SmsFacade) GetAccessor() interface{} {
    return s
}

// Send 发送短信
func (s *SmsFacade) Send(phone string, message string) error {
    return s.service.Send(phone, message)
}

// SendCode 发送验证码
func (s *SmsFacade) SendCode(phone string, code string) error {
    return s.service.SendTemplate(phone, "code", map[string]string{
        "code": code,
    })
}

// SendTemplate 发送模板短信
func (s *SmsFacade) SendTemplate(phone, template string, data map[string]string) error {
    return s.service.SendTemplate(phone, template, data)
}
```

### 6.2 使用示例

```go
// app/controller/auth.go
package controller

import (
    "math/rand"
    "vigo/framework/mvc"
    "vigo/app/facade"
    "vigo/framework/helper"
)

// SendSmsCode 发送短信验证码
func SendSmsCode(c *mvc.Context) {
    phone := c.PostForm("phone")
    
    // 生成验证码
    code := fmt.Sprintf("%06d", rand.Intn(999999))
    
    // 使用短信门面
    err := facade.Sms.SendCode(phone, code)
    if err != nil {
        c.Json(500, map[string]interface{}{
            "code":    500,
            "message": "发送失败",
        })
        return
    }
    
    // 保存验证码到 Redis（5 分钟有效期）
    facade.Redis.Set("sms_code:"+phone, code, 5*time.Minute)
    
    c.Json(200, map[string]interface{}{
        "code":    0,
        "message": "发送成功",
    })
}

// VerifySmsCode 验证短信验证码
func VerifySmsCode(c *mvc.Context) {
    phone := c.PostForm("phone")
    code := c.PostForm("code")
    
    // 从 Redis 获取验证码
    storedCode := facade.Redis.Get("sms_code:" + phone)
    
    if storedCode != code {
        c.Json(400, map[string]interface{}{
            "code":    400,
            "message": "验证码错误",
        })
        return
    }
    
    // 验证成功，删除验证码
    facade.Redis.Del("sms_code:" + phone)
    
    c.Json(200, map[string]interface{}{
        "code":    0,
        "message": "验证成功",
    })
}
```

## 七、门面优势

### 7.1 代码简洁

```go
// 不使用门面
redis := container.Get("redis").(*redis.Client)
redis.Set("key", "value", 0)

// 使用门面
Redis.Set("key", "value", 0)
```

### 7.2 易于记忆

```go
// 所有门面都在 facade 包中
import "vigo/framework/facade"

facade.Redis.xxx
facade.DB.xxx
facade.Cache.xxx
facade.Log.xxx
```

### 7.3 便于测试

```go
// 测试时可以替换门面
func TestPayment(t *testing.T) {
    // 使用 Mock 门面
    mockPayment := &MockPaymentFacade{}
    extend.RegisterFacade(mockPayment)
    
    // 运行测试
    // ...
}
```
