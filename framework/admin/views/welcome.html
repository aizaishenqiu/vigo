Vigo Framework

Vigo 是一个高性能、模块化的 Go 语言 Web 开发框架。它集成了现代微服务架构所需的各种组件，旨在提供极致的开发体验和运行效率。 框架内置了强大的路由引擎、中间件机制、ORM、缓存、消息队列支持以及全链路监控能力，助您快速构建企业级应用。

高性能内核

基于原生 Go 语言特性深度优化，提供极低的内存占用和极高的并发处理能力，轻松应对高并发场景。

微服务架构

原生支持微服务治理，内置服务注册发现（Nacos）、配置中心、负载均衡、熔断降级等核心组件。

极致开发体验

提供完善的 CLI 工具、代码生成器和 Admin 管理后台，统一的代码规范和最佳实践，让开发变得简单愉悦。
