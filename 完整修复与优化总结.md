## 🐛 Bug 修复清单

### 1. 管理面板 404 错误

**问题**: 访问 `http://localhost:8080/admin/` 显示 404

**原因**: 
- 路由只注册了 `/admin`，没有注册 `/admin/`
- 浏览器访问带斜杠的 URL 时无法匹配

**修复**:
```go
// framework/admin/admin.go
r.GET(basePath, indexHandler)
r.GET(basePath+"/", indexHandler)  // 新增
```

**状态**: ✅ 已修复

### 2. 静态资源 404 和 MIME 类型错误

**问题**: 
- CSS/JS 文件返回 404
- MIME 类型显示 `text/plain`，浏览器拒绝加载

**原因**:
- 静态资源处理逻辑使用 `http.FileServer` 不正确
- 没有正确设置 Content-Type

**修复**:
```go
// framework/admin/admin.go
r.GET(basePath+"/static/*filepath", func(c *mvc.Context) {
    filepath := c.Param("filepath")
    filepath = strings.TrimPrefix(filepath, "/")
    
    content, err := adminStatic.ReadFile(filepath)
    if err != nil {
        c.String(404, "File not found: "+filepath)
        return
    }
    
    // 根据文件扩展名设置正确的 Content-Type
    contentType := "application/octet-stream"
    if strings.HasSuffix(filepath, ".css") {
        contentType = "text/css; charset=utf-8"
    } else if strings.HasSuffix(filepath, ".js") {
        contentType = "application/javascript; charset=utf-8"
    }
    // ... 其他类型
    
    c.Writer.Header().Set("Content-Type", contentType)
    c.Writer.WriteHeader(200)
    c.Writer.Write(content)
})
```

**状态**: ✅ 已修复

### 3. ORM 方法名称不习惯

**问题**: 用户习惯使用旧名称方法

**修复**: 添加旧名称别名方法
```go
// framework/db/orm.go

// Column 是 Pluck 的别名
func (o *ORM) Column(column string) ([]interface{}, error) {
    return o.Pluck(column)
}

// Inc 是 Incr 的别名
func (o *ORM) Inc(field string, amount ...int) (int64, error) {
    return o.Incr(field, amount...)
}

// Dec 是 Decr 的别名
func (o *ORM) Dec(field string, amount ...int) (int64, error) {
    return o.Decr(field, amount...)
}

// InsertAll 是 InsertMulti 的别名
func (o *ORM) InsertAll(dataList []map[string]interface{}) (int64, error) {
    return o.InsertMulti(len(dataList), dataList)
}
```

**状态**: ✅ 已修复

```go

    Order("create_time DESC").
    Select()
```

```go

    Where("status", 1).
```
